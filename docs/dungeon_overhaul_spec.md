# 🏰 던전 대개편 기획안 v4.0 — 턴제 수동 전투

## 1. 컨셉

**현재:** 자동배틀, 버프 고르기만 전략 → "뭔가 빠진 느낌"
**목표:** 턴마다 직접 선택, 적의 의도를 보고 판단 → "한 턴이 소중한 로그라이크"

### 핵심 레퍼런스
| 게임 | 차용 요소 |
|------|----------|
| **Into the Breach** | 적 의도 표시 (다음 턴 뭐 할지 미리 보여줌) |
| **슬레이 더 스파이어** | AP(행동력) 시스템, 매 턴 2~3개 선택지 |
| **포켓몬 불가사의의 던전** | PP(기술 횟수), 일반공격 무한 + 특수기 제한 |
| **다키스트 던전** | 긴장감, 실패의 무게감, 연습모드 없음 |

## 2. 전투 플로우

### 2-1. 턴 구조

매 턴 유저에게 보이는 화면:
```
━━━━━━━━━━━━━━━
[26층] 🪨 용암 동굴
━━━━━━━━━━━━━━━
🆚 마그카르고 (바위/불)
❤️ ████████░░ 80%

⚠️ 예고: 화염방사 → 내 포켓몬에게 ~120 데미지!

━━━━━━━━━━━━━━━
🐉 한카리아스 (드래곤/땅)
❤️ ██████░░░░ 62% (310/500)
🔮 상성: 유리! (땅→바위 ×1.5)

[⚔️ 일반공격]  [🐉 용의파동 3/5]
[🪨 지진 2/3]  [🛡️ 방어]
```

### 2-2. 행동 선택지 (4버튼 고정)

| 버튼 | 효과 | 비용 |
|------|------|------|
| ⚔️ **일반공격** | ATK 기반 데미지, 타입 없음 | 무료, 무한 |
| 🔥 **특수기1** | 포켓몬 타입1 기반, 강력 | PP 소모 (런 전체 공유) |
| 💧 **특수기2** | 포켓몬 타입2 기반, 강력 | PP 소모 |
| 🛡️ **방어** | 이번 턴 받는 데미지 50% 감소 | 무료 |

### 2-3. 특수기 PP 시스템

포켓몬 타입에서 자동 배정 (유저가 스킬을 고르지 않음):
- 타입1 → 해당 타입 대표 기술 (이름만, 실제론 타입 데미지)
- 타입2 → 해당 타입 대표 기술
- 단일 타입 포켓몬 → 특수기1만 + 일반공격/방어

**PP 설정 (런 전체 공유, 층 넘어가도 안 참):**

| 희귀도 | 특수기1 PP | 특수기2 PP | 일반공격 배율 |
|--------|-----------|-----------|-------------|
| 일반 | 8 | 8 | ×0.8 |
| 레어 | 7 | 7 | ×0.9 |
| 에픽 | 6 | 6 | ×1.0 |
| 전설 | 5 | 5 | ×1.15 |
| 초전설 | 4 | 4 | ×1.3 |

**의도:** 고코스트 포켓몬은 PP가 적지만 일반공격이 강함. 저코스트는 PP 풍부. 자원 관리 전략.

### 2-4. 적 의도 시스템 (Into the Breach)

적이 다음에 뭘 할지 **미리 표시**:
```
⚠️ 예고: 화염방사 → ~120 데미지!    → 방어할지 공격할지 판단
⚠️ 예고: 방어 태세                    → 공격해도 효과 낮음, 버프 턴?
⚠️ 예고: 힘 모으기 → 다음 턴 강공격!   → 지금 방어? 아님 먼저 죽이기?
```

적 행동 패턴:
| 행동 | 확률 | 설명 |
|------|------|------|
| 일반공격 | 50% | 기본 데미지 |
| 타입공격 | 25% | 적 타입 기반, 상성 적용 |
| 힘 모으기 | 10% | 이번 턴 안 때리고, 다음 턴 ×2 |
| 방어 | 15% | 받는 데미지 50% 감소 |

**보스는 특수 패턴 추가:**
- 5층마다 보스
- 보스 전용 행동: 전체기(무조건 큰 데미지), 회복, 분노(HP 30% 이하 시 공격력 ×1.5)

### 2-5. 데미지 공식 (단순화)

```
일반공격: ATK × 일반배율 × 편차(0.9~1.1)
특수기:   (ATK or SPA) × 1.5 × 타입상성 × 편차
방어 중:  받는 데미지 × 0.5
```

상성 배율 (본가보다 압축):
- 유리: ×1.5
- 불리: ×0.67
- 면역: ×0.3 (던전에서는 완전 면역 없음)

### 2-6. 상성 툴팁

전투 화면에 항상 표시:
```
🔮 상성: 유리! (땅→바위 ×1.5)
💡 특수기 '지진'이 효과적! PP를 아끼지 마세요.
```
```
🔮 상성: 불리 (드래곤→페어리 ×0.67)
💡 방어 위주로, 일반공격으로 버텨보세요.
```

### 2-7. 전투 흐름 (한 턴)

1. **적 의도 표시** + 내 상태 표시
2. **유저 버튼 선택** (⚔️/🔥/💧/🛡️)
3. **속도 비교** → 빠른 쪽 먼저 행동
4. **결과 표시** (데미지, HP 변화, 크리티컬 등)
5. **전투 종료 체크** → 안 끝났으면 다음 턴으로 (메시지 edit)

**한 층 = 3~8턴** (일반), **보스 = 5~15턴**
**한 층 소요시간 = 30초~1분** (버튼 클릭 속도 기준)

### 2-8. 포기 버튼

전투 중 언제든:
```
[⚔️ 일반공격]  [🐉 용의파동 3/5]
[🪨 지진 2/3]  [🏳️ 포기]
```
포기 시 현재까지 도달한 층 기준 보상 지급.

## 3. 층 테마 시스템

### 3-1. 구간별 타입 고정

각 테마(요일별)마다 5층 단위로 적 타입이 변화. 유저가 패턴을 학습 가능.

**화산 테마 예시:**
| 구간 | 적 타입 | 분위기 멘트 |
|------|---------|-----------|
| 1~4층 | 불 100% | "뜨거운 바람이 불어온다... 불꽃 포켓몬의 영역." |
| 5층 | 👑 보스: 불 | "동굴 깊은 곳에서 포효가 울려 퍼진다!" |
| 6~9층 | 불 50% + 바위 50% | "용암이 흐르는 좁은 길... 바위가 길을 막는다." |
| 10층 | 👑 보스: 바위 | "대지가 진동한다..." |
| 11~14층 | 물 70% + 얼음 30% | "차가운 지하수가 스며든다... 상성이 뒤집힌다!" |
| 15층 | 👑 보스: 물 | "거대한 파도가 밀려온다!" |
| 16~19층 | 풀 50% + 독 50% | "화산재 위에 자란 독초의 숲..." |
| 20층 | 👑 보스: 풀/독 | "숲의 지배자가 깨어난다!" |
| 21~24층 | 전기 60% + 강철 40% | "번개가 치는 폐허... 기계 포켓몬이 배회한다." |
| 25층 | 👑 보스: 전기 | "뇌신이 강림한다!" |
| 26~29층 | 드래곤 80% + 페어리 20% | "용의 영역... 그리고 그에 맞서는 자들." |
| 30층 | 👑 보스: 드래곤 | "용왕이 포효한다!" |
| 31~34층 | 에스퍼 + 고스트 | "현실이 일그러진다..." |
| 35층 | 👑 보스: 에스퍼 | |
| 36~39층 | 격투 + 악 | "어둠 속의 투기장..." |
| 40층 | 👑 보스: 격투 | |
| 41~44층 | 혼합 (전 타입) | "모든 것이 뒤섞인 카오스..." |
| 45층 | 👑 보스: 랜덤 전설 | |
| 46~49층 | 혼합 (강화) | "최후의 시련..." |
| 50층 | 👑 최종 보스: 초전설 | |

### 3-2. 다음 구간 예고

5층 보스 클리어 후:
```
✅ 10층 보스 격파!
🎁 보상: 알 즉시부화권 ×1

⚠️ 다음 구간: 💧 지하수맥 (11~14층)
"차가운 지하수가 스며든다... 물 타입이 나타나기 시작!"
💡 물 타입에 대비하세요! 풀/전기 기술이 유효합니다.

[⚔️ 다음 층!]  [🏳️ 여기서 멈추기]
```

유저가 "여기서 멈추기" 선택 시 → 현재까지 보상 확정, 런 종료.
(5층 단위 세이브 포인트)

## 4. 버프 시스템 (유지 + 개선)

기존 버프 고르기는 **그대로 유지** (Fachenko: "스킬 골라서 올라가는 느낌이 핵심").

### 변경점:
- 버프 선택은 **보스 클리어 후** (5층마다)
- 일반 층에서는 전투만
- 버프 3개 중 1개 선택 (기존과 동일)

### 신규 버프 (턴제 대응):
| 버프 | 효과 |
|------|------|
| PP 회복 | 특수기 PP +2 회복 |
| 선제공격 | 속도 무관 항상 선공 |
| 카운터 | 방어 시 받은 데미지의 30% 반사 |
| 관통 | 적 방어 무시 |

## 7. 밸런스 설계

### 7-1. 코스트별 차별화 (수정)

| 코스트 | 희귀도 | 버프 주기 | PP | 일반공격력 | 전략 |
|--------|--------|-----------|-----|----------|------|
| 1~2 | 일반/레어 | 매 보스 | 8/8 | ×0.8~0.9 | PP 풍부, 특수기 많이 사용 |
| 3~4 | 에픽 | 매 보스 | 6/6 | ×1.0 | 균형 |
| 5 | 전설 | 매 보스 | 5/5 | ×1.15 | 강한 일반공격 |
| 6 | 초전설 | 매 보스 | 4/4 | ×1.3 | PP 아끼며 일반공격 위주 |

**버프는 모든 코스트 동일하게 매 보스(5층)마다.** 기존 코스트별 버프 주기 차이를 제거하여 밸런스 역전 해소.

### 7-2. 적 스케일링 (완화)

```
1~10층:  1.0 + floor × 0.02  (1.02 ~ 1.20)
11~20층: 1.20 + (f-10) × 0.025 (1.225 ~ 1.45)
21~30층: 1.45 + (f-20) × 0.03  (1.48 ~ 1.75)
31~40층: 1.75 + (f-30) × 0.04  (1.79 ~ 2.15)
41~50층: 2.15 + (f-40) × 0.05  (2.20 ~ 2.65)
```
기존 대비 후반 스케일링 20% 완화. 50층 도전이 현실적으로 가능하도록.

## 9. 기술적 구현 포인트

### 9-1. 상태 관리
```python
dungeon_state = {
    "user_id": int,
    "run_id": int,
    "floor": int,
    "turn": int,
    "pokemon": {...},       # 내 포켓몬 정보
    "hp": int, "max_hp": int,
    "pp1": int, "pp1_max": int,  # 특수기1 PP
    "pp2": int, "pp2_max": int,  # 특수기2 PP
    "buffs": [...],
    "enemy": {...},         # 현재 적
    "enemy_intent": str,    # 적 다음 행동
    "enemy_hp": int,
    "phase": "battle|buff_select|zone_preview|result",
    "message_id": int,      # edit할 메시지 ID
}
```

### 9-2. 콜백 데이터 형식
```
dg_atk_{user_id}     → 일반공격
dg_sk1_{user_id}     → 특수기1
dg_sk2_{user_id}     → 특수기2
dg_def_{user_id}     → 방어
dg_go_{user_id}      → 다음 층
dg_quit_{user_id}    → 포기
dg_buf_{user_id}_{i} → 버프 선택
dg_stop_{user_id}    → 여기서 멈추기
```

### 9-3. 메시지 edit 패턴
- 전투 중 새 메시지 안 보냄 → 기존 메시지 edit
- 턴마다 텍스트 + 버튼 갱신
- 보스 클리어/층 이동 시에만 새 메시지

### 9-4. 수정 파일
```
services/dungeon_service.py    — 전면 재작성 (턴제 전투, 적 의도, PP)

handlers/dm_dungeon.py         — 전면 재작성 (턴별 콜백 처리)
database/dungeon_queries.py    — 상태 저장 확장
database/item_queries.py       — 신규 아이템 CRUD
database/schema.py             — 마이그레이션
config.py                      — 던전 상수 추가
```
