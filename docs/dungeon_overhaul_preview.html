2. 전투 플로우

2-1. 턴 구조

매 턴 유저에게 보이는 화면:

❤️ ████████░░ 80%

⚠️ 예고: 화염방사 → 내 포켓몬에게 ~120 데미지!

━━━━━━━━━━━━━━━

[🪨 지진 2/3]  [🛡️ 방어]

2-2. 행동 선택지 (4버튼 고정)

버튼 효과 비용
⚔️ 일반공격 ATK 기반 데미지, 타입 없음 무료, 무한
🔥 특수기1 포켓몬 타입1 기반, 강력 PP 소모 (런 전체 공유)
💧 특수기2 포켓몬 타입2 기반, 강력 PP 소모
🛡️ 방어 이번 턴 받는 데미지 50% 감소 무료

2-3. 특수기 PP 시스템

포켓몬 타입에서 자동 배정 (유저가 스킬을 고르지 않음): - 타입1 → 해당 타입 대표 기술 (이름만, 실제론 타입 데미지) - 타입2 → 해당 타입 대표 기술 - 단일 타입 포켓몬 → 특수기1만 + 일반공격/방어

PP 설정 (런 전체 공유, 층 넘어가도 안 참):

희귀도 특수기1 PP 특수기2 PP 일반공격 배율
일반 8 8 ×0.8
레어 7 7 ×0.9
에픽 6 6 ×1.0
전설 5 5 ×1.15
초전설 4 4 ×1.3

의도: 고코스트 포켓몬은 PP가 적지만 일반공격이 강함. 저코스트는 PP 풍부. 자원 관리 전략.

2-4. 적 의도 시스템 (Into the Breach)

적이 다음에 뭘 할지 미리 표시:

⚠️ 예고: 화염방사 → ~120 데미지!    → 방어할지 공격할지 판단
⚠️ 예고: 방어 태세                    → 공격해도 효과 낮음, 버프 턴?
⚠️ 예고: 힘 모으기 → 다음 턴 강공격!   → 지금 방어? 아님 먼저 죽이기?

적 행동 패턴: | 행동 | 확률 | 설명 | |------|------|------| | 일반공격 | 50% | 기본 데미지 | | 타입공격 | 25% | 적 타입 기반, 상성 적용 | | 힘 모으기 | 10% | 이번 턴 안 때리고, 다음 턴 ×2 | | 방어 | 15% | 받는 데미지 50% 감소 |

보스는 특수 패턴 추가: - 5층마다 보스 - 보스 전용 행동: 전체기(무조건 큰 데미지), 회복, 분노(HP 30% 이하 시 공격력 ×1.5)

2-5. 데미지 공식 (단순화)

일반공격: ATK × 일반배율 × 편차(0.9~1.1)
특수기:   (ATK or SPA) × 1.5 × 타입상성 × 편차
방어 중:  받는 데미지 × 0.5

상성 배율 (본가보다 압축): - 유리: ×1.5 - 불리: ×0.67 - 면역: ×0.3 (던전에서는 완전 면역 없음)

2-6. 상성 툴팁

전투 화면에 항상 표시:

🔮 상성: 유리! (땅→바위 ×1.5)
💡 특수기 '지진'이 효과적! PP를 아끼지 마세요.

🔮 상성: 불리 (드래곤→페어리 ×0.67)
💡 방어 위주로, 일반공격으로 버텨보세요.

2-7. 전투 흐름 (한 턴)

  1. 적 의도 표시 + 내 상태 표시
  2. 유저 버튼 선택 (⚔️/🔥/💧/🛡️)
  3. 속도 비교 → 빠른 쪽 먼저 행동
  4. 결과 표시 (데미지, HP 변화, 크리티컬 등)
  5. 전투 종료 체크 → 안 끝났으면 다음 턴으로 (메시지 edit)

한 층 = 3~8턴 (일반), 보스 = 5~15턴 한 층 소요시간 = 30초~1분 (버튼 클릭 속도 기준)

2-8. 포기 버튼

전투 중 언제든:

[⚔️ 일반공격]  [🐉 용의파동 3/5]
[🪨 지진 2/3]  [🏳️ 포기]

포기 시 현재까지 도달한 층 기준 보상 지급.

4. 버프 시스템 (유지 + 개선)

기존 버프 고르기는 그대로 유지 (Fachenko: "스킬 골라서 올라가는 느낌이 핵심").

변경점:

  • 버프 선택은 보스 클리어 후 (5층마다)
  • 일반 층에서는 전투만
  • 버프 3개 중 1개 선택 (기존과 동일)

신규 버프 (턴제 대응):

7. 밸런스 설계

7-1. 코스트별 차별화 (수정)

코스트 희귀도 버프 주기 PP 일반공격력 전략
1~2 일반/레어 매 보스 8/8 ×0.8~0.9 PP 풍부, 특수기 많이 사용
3~4 에픽 매 보스 6/6 ×1.0 균형
5 전설 매 보스 5/5 ×1.15 강한 일반공격
6 초전설 매 보스 4/4 ×1.3 PP 아끼며 일반공격 위주

버프는 모든 코스트 동일하게 매 보스(5층)마다. 기존 코스트별 버프 주기 차이를 제거하여 밸런스 역전 해소.

7-2. 적 스케일링 (완화)

1~10층:  1.0 + floor × 0.02  (1.02 ~ 1.20)
11~20층: 1.20 + (f-10) × 0.025 (1.225 ~ 1.45)
21~30층: 1.45 + (f-20) × 0.03  (1.48 ~ 1.75)
31~40층: 1.75 + (f-30) × 0.04  (1.79 ~ 2.15)
41~50층: 2.15 + (f-40) × 0.05  (2.20 ~ 2.65)

기존 대비 후반 스케일링 20% 완화. 50층 도전이 현실적으로 가능하도록.

9. 기술적 구현 포인트

9-1. 상태 관리

dungeon_state = {
    "user_id": int,
    "run_id": int,
    "floor": int,
    "turn": int,
    "pokemon": {...},       # 내 포켓몬 정보
    "hp": int, "max_hp": int,
    "pp1": int, "pp1_max": int,  # 특수기1 PP
    "pp2": int, "pp2_max": int,  # 특수기2 PP
    "buffs": [...],
    "enemy": {...},         # 현재 적
    "enemy_intent": str,    # 적 다음 행동
    "enemy_hp": int,
    "phase": "battle|buff_select|zone_preview|result",
    "message_id": int,      # edit할 메시지 ID
}

9-2. 콜백 데이터 형식

dg_atk_{user_id}     → 일반공격
dg_sk1_{user_id}     → 특수기1
dg_sk2_{user_id}     → 특수기2

  • 전투 중 새 메시지 안 보냄 → 기존 메시지 edit
