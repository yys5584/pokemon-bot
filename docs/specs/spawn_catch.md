# 스폰/포획 시스템

## 개요
그룹 채팅에서 메시지 활동 기반으로 포켓몬 스폰, ㅊ/ㅁ/ㅎ/ㅊㅊ로 포획.

## 핵심 파일
- `services/spawn_schedule.py` — 일일 스폰 스케줄링
- `services/spawn_execute.py` — 스폰 실행 로직
- `services/spawn_resolve.py` — 포획 결과 처리
- `handlers/group.py` — ㅊ/ㅁ/ㅎ/ㅊㅊ 핸들러
- `database/spawn_queries.py` — 스폰 DB 쿼리
- `config.py` — 모든 상수값

## 스폰 조건
| 항목 | 값 |
|------|-----|
| 최소 인원 | 10명 (`SPAWN_MIN_MEMBERS`) |
| 일일 스폰 | 기본 2회 (채팅방 레벨 보너스 최대 +5) |
| 최대 일일 스폰 | 12회 (`SPAWN_MAX_DAILY`) |
| 스폰 간 간격 | 최소 2시간 (`SPAWN_MIN_GAP_HOURS`) |
| 포획 제한 시간 | 60초 (`SPAWN_WINDOW_SECONDS`) |
| 스폰 쿨타임 | 5분 (`SPAWN_COOLDOWN_SECONDS`) |
| 재시도 간격 | 30~60분 (비활동 시) |

## 포획 메커니즘
| 볼 | 명령 | 포획률 | 비고 |
|----|------|--------|------|
| 일반 | ㅊ/c | 등급별 기본 | 처음 2마리 자동 성공, 일일 20회 |
| 마스터 | ㅁ/m | 100% | 뉴비스폰 불가 |
| 하이퍼 | ㅎ/h | 기본×3.0 | 뉴비스폰 불가 |
| 우선포획 | ㅊㅊ | 100% | 던전 아이템, 뉴비스폰 불가 |

## 경쟁 포획
- 여러 유저 동시 포획 시 **먼저 성공한 사람이 획득**
- 우선순위: 마스터볼 > 하이퍼볼 > 일반볼
- 실패자 마스터볼/하이퍼볼은 환불

## 이로치 확률
| 스폰 타입 | 확률 |
|-----------|------|
| 자연 스폰 | 0% |
| 강제 스폰 | 1% |
| 아케이드 | 1.5% |
| 뉴비 스폰 | 3.3% (1/30) |

- 채팅방 레벨 부스트 추가 가능
- 안티어뷰징: 30분 내 포획 참여자 ≤1명이면 이로치 확률 0%
- 이로치 IV 최소: 10 (`IV_SHINY_MIN`)

## 뉴비 스폰
- **위치**: 아케이드 채팅방 전용
- **확률**: 아케이드 스폰의 20% (`NEWBIE_SPAWN_CHANCE`)
- **도감 티어**: 0(~100), 1(100~200), 2(200~300), 3(300+)
- **Tier 0~2**: 포획 보장 (자동 성공)
- **Tier 3**: 일반 포획률 적용
- **특수볼 불가**: 마스터/하이퍼/우선포획볼 차단

## 아케이드
| 항목 | 값 |
|------|-----|
| 스폰 간격 | 30초 (`ARCADE_SPAWN_INTERVAL`) |
| 티켓 아케이드 간격 | 60초 |
| 포획 윈도우 | 25초 (`ARCADE_SPAWN_WINDOW`) |

## 날씨 부스트
- 배수: 1.5배 (`CAMP_WEATHER_MULTIPLIER`)
- 특정 포켓몬의 스폰 가중치에 적용
- 날씨 종류: 맑음, 비, 폭풍, 안개, 눈, 화창, 바람, 무더위
