# 타로 시스템 — 창백피카츄의 타로

## 개요
DM에서 "타로" 입력 시 6주제 타로 리딩 제공. 78장 카드 × 6주제 × 정/역방향 = 936개 해석문.

## 핵심 파일
- `handlers/dm_fortune.py` — DM 타로 핸들러 (주제 선택, 스프레드 선택, 리딩, 생년월일 입력)
- `services/fortune_service.py` — 카드 뽑기, 스프레드 엔진, 해석 조합, 별자리
- `data/tarot_cards_full.json` — 78장 카드 데이터 + 해석문
- `tests/test_tarot.py` — 39개 테스트

## 동작 흐름
1. DM에서 "타로" 또는 "tarot" 입력
2. 생년월일 미등록 시 → 입력 유도 (건너뛰기 가능)
3. 주제 선택 (연애/직장/재물/투자/인간관계/종합)
4. 투자/연애는 스프레드 선택, 나머지는 쓰리카드 바로 진행
5. 카드 뽑기 → 해석 조합 → 종합 요약 → 메시지 전송
6. DB에 리딩 기록 (통계용)
7. 하루 1회 제한 (user_data 기반)

## 스프레드
| 타입 | 장수 | 포지션 |
|------|------|--------|
| one_card | 1 | 메시지 |
| three_card | 3 | 과거/현재/미래 |
| investment | 4 | 현재 상태/기회/리스크/조언 |
| love | 5 | 내 마음/상대 에너지/관계 현재/장애물/잠재력 |

## DB 테이블
- `users.birth_date` — 생년월일 (DATE, nullable)
- `tarot_readings` — 리딩 기록 (user_id, topic, spread_type, cards_json, reading_date)

## 설정값
- 하루 1회 제한: `context.user_data["last_tarot_date"]`
- 같은 날 같은 유저+주제 = 같은 결과 (시드 기반)

## 생년월일 설정
- 타로 첫 실행 시 입력 유도 (건너뛰기 가능)
- 설정 메뉴(⚙️ 설정 → 🎂 생년월일 설정)에서 변경/삭제 가능
- 형식: YYYY-MM-DD, YYYYMMDD, YYYY/MM/DD, YYYY.MM.DD
