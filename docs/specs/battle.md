# 배틀 시스템

## 개요
6v6 턴제 배틀. 코스트 기반 팀 편성, 타입 상성, IV/성격/친밀도 보정.

## 핵심 파일
- `utils/battle_calc.py` — 데미지 계산, IV, 타입 상성
- `services/battle_service.py` — 배틀 엔진 (`_resolve_battle`)
- `handlers/battle.py` — 배틀 도전/수락/결과 UI
- `handlers/battle_team.py` — 팀 편집/등록/해제
- `config.py` — 모든 상수값

## 데미지 계산
```
base = max(1, (22 × 130 × attack / defense) / 50 + 2)
damage = base × 타입배율 × 크리배율 × 스킬배율 × random(0.9~1.1)
```
- 공격 > 특공이면 공격/방어 사용, 아니면 특공/특방
- 크리티컬: 10% 확률, ×1.5 배율

## 배틀 진행
1. 양 팀 첫 포켓몬 선출
2. 라운드 루프 (최대 50라운드)
   - 선제기 보유 시 30% 발동
   - 속도 높은 쪽 먼저 공격
   - 스킬 효과 (반동/흡수/자폭/나태/잠자기)
   - KO 시 다음 포켓몬 선출
3. 50라운드 초과 시 HP 합산 판정
