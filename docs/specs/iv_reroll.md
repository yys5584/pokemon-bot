# IV 리롤 시스템

## 개요
가챠에서 획득한 아이템으로 포켓몬 개체값(IV)을 재설정. 전체 리롤(6종)과 선택 리롤(1종) 두 가지.

## 핵심 파일
- `handlers/dm_gacha.py` — 리롤 핸들러 (아이템 사용, 포켓몬 선택, 스탯 선택, 실행)
- `database/item_queries.py` — 아이템 CRUD, IV 업데이트
- `utils/battle_calc.py` — `generate_ivs()` IV 생성

## 동작 흐름

### 기존 경로
1. DM "아이템" → 아이템 목록 → IV 리롤 선택
2. 포켓몬 선택 (등급 필터 + 페이지네이션)
3. (선택 리롤) 스탯 선택
4. 실행 → 결과 표시 + 반복 버튼

### 단축 경로
1. DM "리롤" → 보유 아이템 체크 → 리롤 타입 선택 (or 바로 진입)
2. 포켓몬 선택
3. 실행

## 아이템
| 아이템 | 키 | 효과 |
|--------|-----|------|
| 개체값 재설정권 | iv_reroll_all | IV 6종 전부 랜덤 리롤 |
| IV 선택 리롤 | iv_reroll_one | 원하는 스탯 1종만 리롤 |

## UX 개선 (2026-04-02)
- 리롤 완료 후 반복 버튼: "한 번 더" / "다른 포켓몬" / "다른 스탯"
- 남은 아이템 수 표시
- "리롤" 단축 명령어로 아이템 메뉴 스킵

## 콜백 패턴
- `item_use_iv_reroll_all` / `item_use_iv_reroll_one` — 리롤 시작
- `ivr_pk_{mode}_{instance_id}` — 포켓몬 선택
- `ivr_st_{instance_id}_{stat}` — 스탯 선택 (선택 리롤)
- `ivr_stsel_{instance_id}` — 스탯 선택 화면으로 이동
- `ivr_ft_{mode}_{rarity}_{page}` — 등급 필터
- `ivr_pg_{mode}_{rarity}_{page}` — 페이지네이션
