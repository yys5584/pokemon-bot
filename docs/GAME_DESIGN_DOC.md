## 1. 현황 분석

### 1.1 핵심 루프
| 루프 | 주기 | 설명 |
|------|------|------|
| 아케이드 캐치 | 30초 | tg_poke에서 ㅊ 연타. 메인 중독 요소 |
| 일반방 캐치 | 2시간~갭 | 방 활동량 기반 자연 스폰 |

| 교환 | 수시 | P2P 150BP, 교환진화 |

### 1.2 유저 프로필
- **연령/성향**: 2~30대 한국인 크립토 유저
- **특성**: 희소성에 민감, 경쟁심 강함, 플렉스 욕구, 트레이딩 DNA
- **활동 패턴**: 텔레그램 상주, 단톡방 여러 개 동시 참여

### 1.3 핵심 문제
1. **도감 엔드**: 251종 다 모으면 할 게 없음. 그랜드마스터 찍으면 ㅊ 칠 동기 소멸
2. **BP 인플레**: 배틀/토너먼트로 BP 무한 축적 → 마볼 넘침 → 캐치 긴장감 소멸
3. **배틀 단조로움**: 50라운드 택전. 스킬 30% 랜덤이라 전략 깊이 부족
4. **수익화 불가**: 포켓몬 IP 직접 BM 걸면 DMCA 리스크

## 3. Phase 1 상세: 즉시 적용

### 3.1 색이 다른 포켓몬 (Shiny)

**왜 해야 하나**: 도감 251종 완성 후에도 "색다른 버전"을 모으는 동기 부여. 포켓몬 본가에서도 엔드 컨텐츠의 핵심. 크립토 유저 특성상 희소 아이템에 대한 욕구가 매우 높음.

**구현**:
- 스폰 시 1/512 확률로 Shiny 판정 (본가 참고)
- 이벤트로 확률 부스트 가능 (예: 커뮤니티 데이 1/64)
- 카드 이미지에 반짝이 이펙트 + ✨ 이모지
- 별도 Shiny 도감 (251종)
- 포획 메시지: "✨ 색이 다른 ~이(가) 나타났다!"

**밸런스**:
- Shiny는 순수 수집 요소. 배틀 스탯 동일
- 교환 시 Shiny 유지
- 진화 시 Shiny 유지

**기대 효과**: 도감 완성자도 다시 ㅊ 칠 이유 생김. "색다른 뮤츠" 같은 건 교환 가치 폭등.

### 3.2 이벤트 한정몬

**왜 해야 하나**: 시간 한정 희소성 = 크립토 유저의 FOMO 자극. "이때 안 잡으면 영원히 못 잡음"이 핵심.

**구현**:
- pokemon_master에 `is_event` 플래그 + `event_start`, `event_end` 컬럼
- 이벤트 기간에만 스폰 풀에 포함
- 이벤트 종료 후 스폰 불가, 보유한 유저만 소장

**1차 한정몬 후보**:
| 이름 | 이벤트 | 등급 | 비고 |
|------|--------|------|------|
| 흑자몽 | 다크 위크 | Epic | 검은색 리자몽 변종 |
| 산타 피카츄 | 크리스마스 | Rare | 산타모자 피카츄 |
| 만우절 아구몬 | 만우절 | Legendary | 디지몬 콜라보 컨셉 ㅋㅋ |
| 골드 잉어킹 | 황금 주간 | Epic | 금색 잉어킹 |
| 벚꽃 이브이 | 봄 이벤트 | Rare | 분홍 이브이 |

**주의**: 이건 오리지널 변종이라 IP 리스크 낮음. 공식 포켓몬 이름 그대로 쓰면 안 되고 "흑룡" "골드킹" 같은 자체 네이밍 필요할 수도.

### 3.3 도감 보상 강화

**현재 문제**: 도감 채워도 칭호 외에 실질 보상 없음.

**추가 보상**:
| 달성 | 보상 |
|------|------|
| Gen1 50종 | 마볼 3개 |
| Gen1 100종 | 하이퍼볼 20개 + 칭호 |
| Gen1 151종 완성 | 마볼 10개 + 전용 칭호 이모지 |
| Gen2 50종 | 마볼 3개 |
| Gen2 100종 완성 | 마볼 10개 + 전용 칭호 이모지 |
| 251종 그랜드마스터 | 마볼 20개 + 프로필 테두리 |
| Shiny 10종 | 전용 칭호 |
| Shiny 50종 | 전용 칭호 + 프로필 이펙트 |

## 4. Phase 2 상세: 단기

### 4.1 3세대 확장 (호엔 지방)

**규모**: 135종 추가 (252~386)
- 새로운 전설: 그라우돈, 카이오가, 레쿠쟈, 라티오스/라티아스, 레지 3종, 지라치, 데옥시스
- 인기 포켓몬: 가디안, 앱솔, 밀로틱, 메타그로스, 보만다

**도입 방식**:
- "호엔 게이트 오픈" 이벤트 (1주일간 3세대 출현률 UP)
- 3세대 전용 도감 + 칭호 라인 추가
- 3세대 교환진화 추가 (후딘→ 등)

**데이터 작업**: pokemon_master에 135행 추가, 이모지/타입/스탯/진화트리 설정. 카드 이미지 생성.

### 4.2 크루(길드) 시스템

**왜 해야 하나**: 크립토 텔레그램 문화 = 커뮤니티/그룹 정체성이 강함. "우리 방이 제일 잘함" 경쟁이 이미 있음.

**구현**:
- 크루 생성: 방 단위 or 자유 가입 (최대 30명)
- **크루 도감**: 크루원 전체 보유 포켓몬 합산 도감
- **크루 랭킹**: 주간 포획 수, 배틀 승수, 크루 도감 완성도
- **크루 보상**: 주간 1위 크루 → 전원 마볼 1개 + 크루 전용 칭호

**크루 레이드** (4.3 던전과 연계):
- 크루원 여러 명이 동시에 보스몬 공격
- 크루 활동으로만 얻는 한정 보상

### 4.3 랭킹 시즌제

**왜 해야 하나**: 도감 완성 후 경쟁 동기 유지. 시즌 리셋으로 순환 구조 생성.

**시즌 구조**:
- 기간: 1개월 (매월 1일 리셋)
- **시즌 포인트(SP)**: 포획/배틀/토너먼트로 획득
  - 일반 포획: 1 SP
  - 레어 포획: 3 SP
  - 에픽 포획: 10 SP
  - 전설 포획: 30 SP
  - Shiny 포획: 50 SP
  - 배틀 승리: 5 SP
  - 토너먼트 우승: 100 SP
- **시즌 랭킹**: SP 기준 순위
- **시즌 보상**:
  - Top 1: 전용 칭호 (시즌 번호 포함) + 마볼 20 + 한정몬 선택권
  - Top 3: 전용 칭호 + 마볼 10
  - Top 10: 마볼 5
  - Top 50: 마볼 2
- **영구 도감은 유지**: 시즌은 포인트/랭킹만 리셋. 소장 포켓몬은 절대 안 건드림.

## 5. Phase 3 상세: 중기

### 5.1 배틀 리워크

**현재 문제**: 스킬 30% 랜덤 발동, 전략적 선택지 없음.

**개선안: 스킬 선택제**
- 포켓몬마다 2~3개 스킬 보유
- 턴마다 스킬 선택 (시간 제한 10초, 미선택 시 기본공격)
- 스킬 종류: 공격/방어/버프/디버프
- PP(사용 횟수) 제한으로 자원 관리 요소

**랭크 매치**:
- 래더 시스템: 브론즈 → 실버 → 골드 → 플래티넘 → 다이아 → 마스터
- 시즌 종료 시 보상 + 소프트 리셋
- 래더 점수(ELO) 기반 매칭

### 5.2 던전/레이드

**왜 해야 하나**: PvP만으로는 한계. 협동 PvE = 커뮤니티 결속 + 새로운 보상 루트.

**구현 컨셉**:
- 아케이드 채널에 "던전 입장" 시간대 (예: 매일 20시)
- 보스몬 출현 (HP 10,000 같은 큰 수치)
- 참가자 전원이 자기 팀으로 턴제 공격
- 보스 처치 시 참여도 비례 보상 (BP, 한정 아이템)
- 보스 드랍: 해당 보스의 포켓몬 (레이드 전용 마크 부착)

**레이드 보스 예시**:
- 월요일: 뮤츠 (에스퍼)
- 수요일: 루기아 (비행)
- 금요일: 칠색조 호오우 (불꽃)
- 일요일: 랜덤 전설

### 5.3 프로필 커스텀

**현재**: 칭호 1개 + 이모지 1개만 표시.

**확장**:
- 프로필 카드 (배경색/테두리/배지)
- 대표 포켓몬 3마리 진열
- 업적 배지 시스템 (배틀 100승, 전설 10마리 등)
- **이건 BM 가능 영역**: 프로필 꾸미기 = IP 무관

## 6. Phase 4: 장기 방향

### 6.1 IP 리스크 해소 전략

**현실**: 포켓몬 IP를 사용하는 한 Stars/유료 BM은 위험.

**선택지**:

**A안. 오리지널 전환**
- 자체 몬스터 디자인으로 점진적 교체
- "3세대"부터 오리지널로 시작 → 기존 1~2세대도 순차 교체
- 장점: BM 자유, 법적 안전
- 단점: 포켓몬 팬 이탈 가능성, 디자인 비용

**B안. 현행 유지 (비영리)**
- 포켓몬 IP 그대로, BM 안 함
- 서버비만 후원으로 충당
- 장점: 팬 유지, 법적 리스크 최소화
- 단점: 수익 없음

**C안. 하이브리드**
- 기존 포켓몬 = 무료 컨텐츠
- 오리지널 몬스터 = 유료 컨텐츠 (Stars)
- 프로필 꾸미기/이모지 = 유료 (IP 무관)
- 장점: 점진적 전환 가능
- 단점: 세계관 혼재

**추천: C안** → 오리지널 한정몬부터 시작해서 반응 보고 확대

### 6.2 BM 설계 (오리지널 IP 전제)

| 상품 | 가격 | 설명 |
|------|------|------|
| 프리미엄 프로필 | 월 500원 | 프로필 카드 커스텀, 테두리, 배지 |
| 한정몬 소환권 | 1,000원 | 이벤트 한정 오리지널 몬스터 1회 도전 |
| 시즌 패스 | 2,000원/월 | 시즌 미션 보상 2배 + 전용 칭호 |
| 이모지 팩 | 500원 | 프로필/채팅용 커스텀 이모지 |

Stars 결제 → 텔레그램 30% 수수료 감안하여 가격 설정.

## 8. 크립토 유저 특화 요소

이 유저층에 맞는 설계 원칙:

1. **희소성 설계**: Shiny 1/512, 한정몬, 시즌 한정 칭호 → "안 잡으면 영영 못 잡음"
2. **경쟁 가시화**: 랭킹, 시즌 순위, 크루 대전 → "내가 몇 등인지" 항상 보임
3. **플렉스 수단**: 프로필 커스텀, 희귀 칭호, Shiny 컬렉션 → "나 이거 있음" 자랑
4. **트레이딩 욕구**: 교환 시스템 이미 있음. Shiny/한정몬 추가되면 거래 수요 폭증
5. **FOMO 활용**: 시간 한정 이벤트, 시즌 종료 카운트다운, 한정몬 마감
