> 날짜: 2026-03-18
> 목적: dungeon_spec.md 설계 보강을 위한 실제 게임 메카닉 레퍼런스

## 1. 버프 카테고리 — 흥미로운 선택을 만드는 구조

### 1-1. Vampire Survivors: "무기 + 패시브 = 진화" 시스템

핵심: 단순 스탯업이 아닌 **조합 진화**가 리플레이를 만든다.

- 레벨업 시 **3~4개 중 택1** (무기/패시브 혼합)
- 각 아이템에 **가중치(rarity)** 존재, 중복 제시 없음
- PowerUp으로 **Reroll** (선택지 재굴림) / **Skip** (경험치 보존) 가능
- **무기 진화 조건**: 무기 Lv.8 + 특정 패시브 아이템 + 10분 이후 보스 상자
  - 예: 도끼 + 촛대 = 데스 스파이럴
  - 예: 불 지팡이 + 시금치(공격력%) = 헬파이어
- **Union**: 무기 2개가 합쳐져 1개 슬롯으로 → 슬롯 절약
- **Gift**: 조건 충족 시 추가 아이템 획득 (기존 무기 유지)

**텔레그램 봇 적용 아이디어:**
```
공격 버프 "화염 강화" + 유틸 버프 "타입 일치 부스트"
→ 10층 이후 합체: "화염 폭주" (타입 일치 데미지 2배 + 화상 효과)
```
- 개별로는 무난한 버프가 **조합하면 전설급으로 진화**
- 플레이어가 "이 버프 나오면 저걸 기다려야지" 하는 **전략적 기다림** 생성

### 1-2. Slay the Spire: 유물(Relic) 시너지 설계

핵심: **영구 패시브 유물**이 매 런의 전략 방향을 결정.

- 유물 등급: Common / Uncommon / Rare / Boss (50%/33%/17% 비율)
- 유물은 **영구 버프** — 획득하면 런 내내 유지
- 주요 유물 예시 (구체적 수치):
  - 전투 종료 시 HP 6 회복
  - 전투 시작 시 카드 2장 추가 드로우
  - 전투 시작 시 Block 10 획득
  - 적 Block 50% 감소
  - HP 50% 이하 시 공격력 2배 (1턴)
- **보스 유물**: 극단적 트레이드오프
  - "에너지 +1 대신 최대 HP -10%"
  - "카드 드로우 +2 대신 무작위 카드 1장 Exhaust"

**텔레그램 봇 적용 아이디어:**
```
보스 버프 (10층 관장 클리어 시):
  [⚡ 유리 대포: 공격 +50%, 방어 -30%]
  [🐢 철벽:    방어 +50%, 공격 -20%]
  [🎲 도박꾼:  버프 등급 2단계↑, HP 20% 즉시 감소]
```
- 트레이드오프 있는 선택 = 모든 옵션이 매력적 = 진정한 선택

### 1-3. 버프 카테고리 분류 체계 (권장)

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| **직접 스탯** | 단순 수치 증가 | 공격+15%, HP+20% |
| **조건부** | 특정 상황에서만 발동 | HP 50% 이하 시 공격 2배 |
| **시너지** | 다른 버프와 조합 시 보너스 | "화염" 2개 모으면 화상 추가 |
| **트레이드오프** | 한쪽 올리면 한쪽 내림 | 공격+50% 방어-30% |
| **유틸리티** | 게임 규칙 변경 | 버프 선택지 4개로 증가, 상점 등장 |
| **생존** | 체력/회복 관련 | 매턴 3% 회복, 1회 부활 |
| **상성 조작** | 타입 관계 변경 | 불리 상성 무효, 유리 상성 데미지 1.5배 |

**핵심 원칙**: "스탯+15%" 3개 중 택1은 재미없다. **카테고리가 다른** 3개 중 택1이 최고.

## 2. 층수별 난이도 스케일링 — 실제 게임 수치

### 2-1. PokeRogue: 웨이브 + 토큰 시스템

- 50웨이브마다 적에게 **Damage Token / Protection Token / Recovery Token** 부여
- 각 토큰은 영구 버프 (스택)
  - Damage Token: 적 데미지 증가
  - Protection Token: 적 받는 데미지 감소
  - Recovery Token: 적 매턴 회복
- 타입 강화 아이템: 1개당 해당 타입 데미지 **+20%**, 스택 가능 (4개 = +80%)
- 비타민류: 1개당 특정 스탯 **+10%**, IV 수만큼 최대 스택
- Soul Dew: 10스택 시 유리 스탯 +110%, 불리 스탯 -100% (1로 고정)

**적용 가능한 스케일링 공식:**
```python
# 현재 spec: 적스탯 = 기본 × (1 + 층수 × 0.08)
# → 50층에서 5.0배 — 너무 선형적

# 제안: 구간별 가속 (PokeRogue 토큰 영감)
def enemy_scaling(floor):
    base = 1.0
    if floor <= 10:
        return base + floor * 0.06      # 1~10: 1.06~1.60 (입문)
    elif floor <= 20:
        return base + 0.60 + (floor - 10) * 0.10  # 11~20: 1.70~2.60 (중반)
    elif floor <= 30:
        return base + 1.60 + (floor - 20) * 0.15  # 21~30: 2.75~4.10 (후반)
    else:
        return base + 3.10 + (floor - 30) * 0.20  # 31+: 4.30+ (극한)

# 결과:
#  5층: 1.30  |  10층: 1.60  |  15층: 2.10
# 20층: 2.60  |  25층: 3.35  |  30층: 4.10
# 35층: 5.10  |  40층: 6.10  |  50층: 8.10
```

### 2-2. Summoners War ToA: 에너지 비용도 함께 스케일링

- ToA Normal/Hard: 100층, 10층마다 보스
- **에너지 비용**: 초반 3 → 20층마다 +1씩 증가 (총 100층 = 500 에너지)
- ToA Hell: 10층만, 각 층에 **제약 조건(Objective)** 부여
  - "적이 CC 면역" / "회복 불가" / "치명타 데미지 +200%"
  - 제약 조건 많이 켤수록 별(★) 더 획득 → 보상 증가
- **매월 15일 리셋** → 월 1회 반복 콘텐츠

**텔레그램 봇 적용 아이디어 — "도전 모드" (Hell 영감):**
```
🏰 던전 도전 모드 (주 1회)
난이도 옵션 선택:

[☠️ 적 CC 면역] [💔 회복 불가] [⚡ 적 선공 고정]
[🔥 상성 불리 2배] [💀 버프 절반 효과]

선택한 제약 수에 따라 보상 배율:
0개: ×1.0 | 1개: ×1.5 | 2개: ×2.0 | 3개: ×3.0
```

### 2-3. AFK Arena Arcane Labyrinth: 유물로 난이도 상쇄

- 3층 구성, 매 전투 후 유물 **3개 중 택1**
- 유물 등급: Rare(파랑) / Epic(보라) / Legendary(금)
- 핵심 유물 수치:
  - 체력 재생 +80%
  - 특정 진영 쿨타임 -30%, 에너지 초당 +30
  - 회피율 +20/40% (×1.2/×1.4 계수)
  - **전투 시작 시 적 5초 행동불능**
  - **적 HP 35% 감소 (10초에 걸쳐 복구)**
- 유물 효과 **스택 가능** → "공격 유물 몰빵" 전략이 최강
- 48시간마다 리셋

**핵심 인사이트**: 스케일링이 가파를수록 "버프 빌드"의 가치가 올라감.
선형 스케일링 → 아무 버프나 OK. 가속 스케일링 → 시너지 빌드만 생존.

## 3. 일일/주간 보상 루프 — 습관 형성 설계

### 3-1. 모바일 게임 리텐션 벤치마크 (2025~2026)

- D1 리텐션: 평균 26%, 경쟁력 있으려면 **30~35%**
- D7 리텐션: 평균 10%, 목표 **15~18%**
- D30 리텐션: **4% 미만** (대부분)
- 길드/얼라이언스 참여자: **비참여자 대비 플레이타임 3~4배**
- 60% 이상의 플레이어가 D3 전에 얼라이언스 가입해야 D7 리텐션 유지

### 3-2. 효과적인 일일 콘텐츠 루프 구조

**Summoners War 모델:**
```
매일: 무료 입장 1회 (ToA는 에너지만 있으면 무제한)
매주: ToA 리셋 없음 (월간)
매월: ToA Normal + Hard 리셋 → 300 크리스탈 + 스크롤 다수
```

**Cookie Run Kingdom 모델:**
```
매일: 길드 보스 최대 9회 공격 (회당 60초)
매주: 라운드 보상 + 참여 보상 (18회 출전 시 400 메달)
매달: 시즌 보상 (15회 이상 참여 조건)
```

**텔레그램 봇 최적 루프 (제안):**
```
━━━ 일일 ━━━
  무료 입장권 1장 (자정 리셋)
  구독자: +1장
  → "오늘 던전 안 돌았네" → 매일 접속 동기

━━━ 주간 ━━━
  주간 랭킹 리셋 (월요일 00:00 KST)
  3회 이상 도전 시 "주간 참여 보상" 지급
  → 최소 3일은 접속

━━━ 월간 / 시즌 ━━━
  시즌 (4주) 최고층 기록으로 "시즌 칭호" 부여
  시즌 누적 도전 횟수 마일스톤 보상:
    10회: 입장권 3장
    25회: 결정 2개
    50회: 마스터볼 1개 + 시즌 전용 칭호
  → 꾸준히 해야 마스터볼
```

### 3-3. 연속 접속 (스트릭) 보상

```
1일: 입장권 1장
3일: BP 100
5일: 조각 랜덤 3개
7일: 입장권 2장 + 결정 1개
14일: 입장권 3장 + 결정 2개
30일: 마스터볼 1개

※ 중간에 끊어도 처음부터가 아닌 "이전 단계"로 후퇴
   (예: 14일째에 놓치면 → 7일 단계로 복귀)
```

## 4. 입장권/스태미나 시스템 — F2P vs 유료 밸런스

### 4-1. 업계 일반 패턴

| 게임 | 스태미나 구조 | F2P 지급 | 유료 |
|------|-------------|---------|------|
| Summoners War ToA | 에너지 (3~7/층) | 자연 회복 (6분/1) | 크리스탈로 충전 |
| AFK Arena Lab | 48시간 리셋 | 무료 무제한 | 없음 (유물 구매) |
| Cookie Run Kingdom | 1라운드 9회 | 무료 전부 | 없음 |
| Blue Archive | AP (6분/1 회복) | 레벨업 시 풀충전 | 크리스탈로 구매 |
| 일반 모바일 | 에너지 제한 | 일일 N회 무료 | 충전 or 광고 시청 |

### 4-2. "보상 감소" 모델 (추천)

일부 게임에서 사용하는 대안 — **플레이는 무제한, 보상만 감소**:
```
1~2회: 보상 100%
3회: 보상 70%
4회: 보상 40%
5회+: 보상 10% ("연습 모드")
```
- F2P: "못하게 막힌다" 불만 없음
- 과금자: 풀보상으로 더 많이 돌리는 이점
- **랭킹은 최고층 기록이므로 연습 무제한 = 실력 향상 동기**

### 4-3. 구독자 차별화 (현재 spec 보강)

```
           무료       베이직 구독    채널장 구독
입장권:     1/일        2/일          3/일
BP 구매:   50BP/장     40BP/장       30BP/장
일일 한도:  3장         5장           7장
버프 Reroll: 불가       1회/런        3회/런  ← 핵심 차별점
보상 배율:  ×1.0       ×1.2          ×1.5
```

## 5. "매번 다른 런" — 리플레이 요소

### 5-1. 적 다양성 만드는 법

**PokeRogue**: 바이옴(지형)에 따라 등장 포켓몬 풀 변경
**Pokemon Mystery Dungeon**: 매 입장마다 맵/아이템/적 위치 완전 랜덤

**텔레그램 봇 적용 — "던전 테마" 시스템:**
```
매일 던전 테마가 랜덤 변경:

🔥 화산 던전: 불 타입 적 비중 ↑, 물 타입 버프 보너스
🌊 심해 던전: 물 타입 적 비중 ↑, 전기 타입 버프 보너스
🌿 고대 숲:  풀/독 타입 적 비중 ↑, 비행 타입 버프 보너스
⚡ 발전소:  전기/강철 타입 적 비중 ↑, 땅 타입 버프 보너스
👻 유적지:  고스트/에스퍼 적 비중 ↑, 악 타입 버프 보너스
🐉 용의 둥지: 드래곤 타입 적 비중 ↑ (주간 특수, 금요일만)
```
- 테마에 따라 **최적 포켓몬이 바뀜** → 다양한 포켓몬 육성 동기
- "오늘 화산이네, 갸라도스로 가야지" → 매일 다른 전략

### 5-2. 층 변형자 (Floor Modifier)

**5층마다 보스 대신, 층마다 소규모 변형:**
```
일반 층:   적 1마리 (70% 확률)
엘리트 층: 적 1마리 강화판 (20% 확률, 스탯 ×1.3)
이벤트 층: 선택 이벤트 (10% 확률)

이벤트 예시:
  🏪 상점: BP 사용해 버프 구매 / HP 회복
  🎁 보물: 무조건 버프 1개 추가 획득
  ❓ 도박: HP 30% 걸고 → 성공 시 전설 버프, 실패 시 HP 감소
  🔄 변환: 현재 버프 1개를 상위 등급으로 업그레이드
  ⚔️ 정예: 보스급 적, 이기면 버프 2개 선택
```

### 5-3. 버프 조합이 런을 결정하게 만드는 법

**Slay the Spire 원칙**: "모든 것이 모든 것과 상호작용"

```
버프 시너지 예시:

"화염 강타" (불 타입 데미지 +25%)
  + "잔불"   (화상 상태이상 부여, 매턴 5% 데미지)
  = 시너지 발동: "업화" — 화상 데미지 3배, 중첩 가능

"철벽"     (방어 +30%)
  + "반격"   (받은 데미지 20% 반사)
  = 시너지 발동: "카운터" — 반사 데미지 50%, 공격 시 자체 방어 무시

"속공"     (스피드 +25%)
  + "연속타" (2연타 20% 확률)
  = 시너지 발동: "질풍" — 2연타 확률 50%, 스피드 비례 추가 데미지
```

**구현 포인트:**
- 시너지 조합을 **미리 공개하지 않음** → 발견하는 재미
- 하지만 버프 설명에 힌트 포함: "화염 강타 — 불꽃 계열과 함께하면..."
- 시너지 발견 시 **도감에 등록** → 수집 요소

## 6. 타입 상성을 랜덤 던전에서 다루는 법

### 6-1. Pokemon Emerald Rogue 접근법

- 관장 순서 랜덤, 상대 타입을 **직전까지 모름**
- 루트가 적을수록 팀 다양성 확보 어려움 → 상성 불리 위험 ↑
- **긴장감 원천**: "내 불 타입에 물 관장이 나오면..."

### 6-2. Pokemon Mystery Dungeon 접근법

- 타입 상성 데미지 배율: 유리 ×1.4, 불리 ×0.7 (본편보다 완화)
- 이중 타입 유리: ×1.96 (본편 ×4 대비 훨씬 완만)
- **던전 내 타입 불리를 극복할 수 있는 다양한 기술** 보유 가능

### 6-3. 텔레그램 봇 적용 — 상성 완화 + 버프 대응

```
기본 상성 배율 (배틀과 동일):
  유리: ×1.5 데미지
  불리: ×0.67 데미지
  면역: ×0.0 (→ 던전에서는 ×0.3으로 완화)

상성 관련 버프:
  🔄 "상성 경감"    — 불리 배율 0.67 → 0.85
  🌈 "무속성 일격"  — 상성 무시 공격 30% 확률
  ⚡ "약점 간파"    — 유리 배율 1.5 → 2.0
  🛡 "속성 방벽"    — 불리 추가 데미지 50% 감소

핵심 설계:
  - 상성 불리가 나와도 "버프로 극복할 수 있었는데" 느낌
  - 완전 면역(×0) 대신 ×0.3 → "어렵지만 불가능하진 않음"
  - 상성 유리 연속 = 고층 치고 올라가는 쾌감
  - 상성 불리 연속 = "다음 판엔 나을 거야" (재도전 동기)
```

### 6-4. 테마 던전에서의 상성 예측 가능성

```
화산 던전이면:
  "불 타입 적이 많겠지" → 물 포켓몬 선택 → 유리
  BUT 5층 보스가 풀/땅 타입일 수 있음 → 방심 불가

이렇게 하면:
  - 완전 랜덤보다 전략적 (포켓몬 선택에 의미)
  - 완전 예측 가능보다 긴장감 (보스는 테마 벗어날 수 있음)
```

## 7. 마일스톤 보상 — 장기 목표 설계

### 7-1. Summoners War ToA 보상 구조 (상세)

```
ToA Normal (100층):
  5층:   소환석 1개
  10층:  크리스탈 30 + 소환석 1개
  15층:  소환석 1개
  20층:  크리스탈 30 + 소환석 1개
  30층:  신비 스크롤 1장
  40층:  크리스탈 30
  50층:  신비 스크롤 2장
  60층:  크리스탈 30
  70층:  크리스탈 30
  80층:  크리스탈 30
  90층:  크리스탈 30
  100층: 전설 스크롤 1장 + 빛/어둠 스크롤 1장

ToA Hard 추가 보상:
  동일 구조 + 상위 보상 (전설 스크롤 2장 등)

ToA Hell:
  별(★) 기반: 21~30별 → 별당 크리스탈 50
  제약 조건 더 켤수록 별 더 획득
```

### 7-2. Cookie Run Kingdom 길드 보스 마일스톤

```
라운드 내 18회 전투 완료: 영웅 메달 400
길드 보스 레벨 올릴수록: 레벨당 보상 증가
시즌 15회 이상 참여: 시즌 보상 수령 자격
길드 랭킹: 트로피 → 순위별 시즌 보상
```

### 7-3. 텔레그램 봇 마일스톤 (구체적 제안)

**A. 개인 최고층 마일스톤 (1회성, 영구)**
```
5층 돌파:   칭호 "던전 입문자" + BP 100
10층 돌파:  칭호 "던전 탐험가" + 입장권 3장
20층 돌파:  칭호 "던전 정복자" + 결정 3개
30층 돌파:  칭호 "던전 마스터" + 결정 5개
40층 돌파:  칭호 "던전 레전드" + 무지개 조각 1개
50층 돌파:  칭호 "던전 챔피언" + 마스터볼 1개

★ 칭호에 최고층 숫자 표시: "던전 챔피언 [52F]"
```

**B. 누적 도전 횟수 마일스톤 (시즌 리셋 없음)**
```
10회:   BP 200
25회:   입장권 3장
50회:   결정 2개 + 칭호 "단골 도전자"
100회:  결정 5개 + 칭호 "던전 중독자"
250회:  무지개 조각 2개
500회:  마스터볼 1개 + 칭호 "던전 폐인"
1000회: 전용 이모지 + 칭호 "던전의 왕"
```

**C. 포켓몬별 최고층 기록**
```
각 포켓몬으로 달성한 최고층 기록 저장
→ "이 포켓몬으로 30층 가봤어?" → 다양한 포켓몬 사용 유도
→ 포켓몬 도감에 던전 기록 표시

전 포켓몬 10층 달성: 칭호 "만능 트레이너"
전 포켓몬 20층 달성: 칭호 "전설적 트레이너"
```

**D. 시즌 (4주) 마일스톤**
```
시즌 참여 10회: 조각 랜덤 5개
시즌 참여 25회: 결정 3개
시즌 최고층 Top 10: 시즌 전용 칭호 (예: "S1 던전 엘리트")
시즌 최고층 1위: 특수 칭호 + 마스터볼 (시즌 칭호는 다음 시즌까지 유지)
```
