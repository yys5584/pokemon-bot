# 주간보스 v2 — 크래시 게임 + 영혼 시스템

## 개요
매주 보스 포켓몬 등장. 유저가 GO/STOP으로 딜량 배율을 올리는 크래시 게임.
터지면 포켓몬이 영혼 상태 → 부활 필요 (과금/광고).

## 핵심 구조

### 전투 흐름
1. DM에서 `보스` → 보스 정보 + 보스팀 설정
2. `⚔️ 공격!` → 첫 포켓몬 투입, 기본 딜 적립
3. 매턴 선택:
   - **🔥 GO** → 배율 증가 + 추가 딜. 근데 터질 확률↑
   - **💰 STOP** → 현재 딜량 확정, 보상 수령
4. 터지면 → 해당 포켓몬 **영혼** + 딜량 50% 손실
5. 다음 포켓몬 자동 투입 → 다시 GO/STOP
6. 6마리 전멸 or STOP → 종료

### 배율 & 터질 확률
| GO 횟수 | 배율 | 터질 확률 | 위험도 |
|---------|------|----------|--------|
| 0 (기본) | x1.0 | 0% | 🟢🟢🟢🟢🟢 |
| 1 | x1.5 | 5% | 🟢🟢🟢🟢🟡 |
| 2 | x2.2 | 12% | 🟢🟢🟢🟡🟡 |
| 3 | x3.0 | 22% | 🟢🟢🟡🟡🔴 |
| 4 | x4.0 | 35% | 🟢🟡🟡🔴🔴 |
| 5 | x5.5 | 50% | 🟡🟡🔴🔴🔴 |
| 6+ | x8.0+ | 70%+ | 🔴🔴🔴🔴🔴 |

※ 포켓몬별 독립 (포켓몬 교체되면 GO 횟수 리셋)

### 터졌을 때 (CRASH)
- 현재 포켓몬 → **영혼 상태** (배틀/던전/거래 사용 불가)
- 누적 딜량의 50% 손실
- 다음 포켓몬 자동 투입
- 6마리 전멸 → 강제 종료 (남은 딜량으로 보상)

### STOP
- 현재 딜량 확정
- 보스 공유 HP 차감
- 마일스톤 보상 즉시 지급
- 포켓몬 안전 (영혼 안 됨)

## 영혼 시스템

### 영혼 상태
- 보스전에서 터진 포켓몬 → `soul_status = 'dead'`
- 배틀팀/던전/교환/거래 사용 불가
- 도감/상태창에 💀 표시

### 부활 방법
1. **대시보드 방문** (무료, 1일 1회)
   - 봇에서 `tgpoke.com/revive?token=xxx` 링크 제공
   - 페이지 방문 시 자동 부활 (파트너 배너 노출)
   - 토큰: 1회용, 24시간 만료
2. **마스터볼** (즉시)
   - 마볼 1개로 즉시 부활
3. **BP** (등급별 차등)
   - 일반 100 / 레어 200 / 에픽 500 / 전설 1000 / 초전설 2000
