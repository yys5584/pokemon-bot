📋 개요

매주 등장하는 강력한 보스 포켓몬을 전체 유저가 협력해서 공격하는 비동기 레이드.
하루 1회 공격, 딜량 기록 → 데일리 마일스톤 + 주간 랭킹 보상.

🐉 보스

등장 방식

  • 매주 월요일 자동 생성
  • 전설/초전설 포켓몬 중 랜덤 (직전 주와 중복 방지)
  • 속성은 해당 포켓몬의 원래 속성

보스 스탯

❤️ HP500,000 (조정 가능)
⚔️ 공격/방어초전설급 × 높은 배율
🌿 속성원작 상성 그대로 적용

HP 세팅 근거

  • 초전설 풀강 팀 → 1회 딜량 ~3,000~5,000
  • 일반 12코 팀 → 1회 딜량 ~1,500~3,000
  • DAU 20명 × 7일 = 140타 → 예상 총딜 350K~700K
  • 500,000이면 간신히 잡히는 수준

⚔️ 전투

방식

  • 던전 자동전투 엔진 재활용
  • 내 배틀팀 6마리가 순서대로 보스와 싸움
  • 전멸 or 턴 제한(60턴)까지의 총 딜량 기록
  • 상성 유리한 팀 구성이 핵심 전략

보스 행동

  • 매턴 공격 (일반 / 속성 / 풀어택 랜덤)
  • 방어·회복 없음 샌드백
