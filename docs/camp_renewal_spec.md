# 캠프 리뉴얼 기획서

## 변경 요약

| 항목 | 기존 | 변경 |
|------|------|------|
| 배치 타이밍 | 3시간마다 30분 접수 윈도우 | **상시 배치** (언제든 배치/교체) |
| 정산 | 라운드 시작 시 이전 라운드 정산 | 유지 (3시간마다 정산) |
| 거점 지정 | 없음 (채팅방에서만 조작) | **거점 1개** (구독자 2개) |
| 접근 방식 | 채팅방 "캠프"만 가능 | **DM + 채팅방 양방향** |
| 알림 | 그룹 공지만 | **DM으로 라운드 조건 안내** |
| 거점 변경 | - | **1주일 쿨다운** |

## 1. 거점캠프 시스템

### 1-1. 거점 지정

- 유저가 채팅방에서 **"캠프등록"** 또는 DM에서 **캠프 목록 → 선택**으로 거점 지정
- 유저당 **거점 1개** (구독자는 **2개**까지 가능)
- 거점 변경 쿨다운: **7일** (지정 후 7일간 변경 불가)

### 1-2. DB 변경

```sql
-- users 테이블에 추가
ALTER TABLE users ADD COLUMN home_camp_chat_id BIGINT;       -- 거점 채팅방 ID
ALTER TABLE users ADD COLUMN home_camp_set_at TIMESTAMPTZ;    -- 거점 설정 시각 (쿨다운 계산용)
```

### 1-3. DM "거점캠프" 명령

DM에서 "거점캠프" 입력 시 표시되는 정보:

```
🏕 거점캠프 — [크립토라운지]
👉 t.me/crypto_lounge  (채팅방 링크)
━━━━━━━━━━━━━━━
📊 캠프 레벨: Lv.3
🌿 숲 — 이상해꽃 배치 중 (4점)
🔥 화산 — 비어있음
💧 호수 — 갸라도스 배치 중 (2점)
━━━━━━━━━━━━━━━
📋 다음 정산: 15:00 (1시간 23분 후)

🔄 라운드 보너스 조건:
  🌿 숲: 이상해씨 (특공 15↑) → 7점
  🔥 화산: 리자몽 (공격 14↑) → 7점
  💧 호수: 갸라도스 (방어 13↑) → 7점
━━━━━━━━━━━━━━━
[배치하기]  [거점변경] (3/14 이후 가능)
```

**핵심 포인트:**
- 현재 뭘 배치해야 좋은지 **조건 + 점수를 한 눈에**
- 채팅방 링크 표시 (Telegram invite link 또는 t.me 링크)
- 다음 정산까지 남은 시간 카운트다운
- 거점 미설정 시 → "거점캠프를 설정하세요!" + 활성 채팅방 목록 표시

### 1-4. 채팅방 링크

- `chat_rooms` 테이블에 `invite_link` 컬럼 추가
- 봇이 채팅방에 입장/활성화 시 `get_chat().invite_link` 저장
- 없으면 `export_chat_invite_link()` 시도 (봇이 관리자인 경우만)
- 링크 없으면 채팅방 이름만 표시

```sql
ALTER TABLE chat_rooms ADD COLUMN invite_link TEXT;
```

## 2. 상시 배치 시스템

### 2-1. 기본 흐름

```
[유저] 아무 때나 "캠프" (채팅방) 또는 DM → 포켓몬 배치/교체
  ↓
[시스템] 3시간마다 자동 정산 → 배치된 포켓몬 기준 점수 계산
  ↓
[시스템] 정산 후 다음 라운드 보너스 조건 공개
  ↓
[유저] 조건 보고 배치 교체 (다음 정산 전까지 자유)
```

### 2-2. 변경 포인트

| 기존 | 변경 |
|------|------|
| `CAMP_ACCEPT_WINDOW = 1800` (30분) | 삭제 — 상시 배치 |
| 라운드 시작 시에만 배치 가능 | 언제든 배치/교체 가능 |
| 배치 후 교체 불가 | 교체 가능 (일일 배치 횟수 차감) |

### 2-3. 어뷰징 방지

**옵션 A: 최소 유지 시간** (권장)
- 배치 후 **1시간 이상** 유지해야 정산 점수 인정
- 정산 1시간 전에 몰아서 바꾸는 것 방지
- `placed_at` 필드로 체크: `now() - placed_at >= 1시간`

**옵션 B: 시간 비례 점수**
- 배치 시간에 비례하여 점수 계산 (3시간 전 = 100%, 1시간 전 = 33%)
- 복잡도가 높아 A 권장

### 2-4. 일일 배치 횟수

기존 도감 수 기반 테이블 유지:

| 도감 수 | 일일 배치 횟수 |
|---------|---------------|
| 0~50 | 2회 |
| 51~100 | 3회 |
| 101~200 | 4회 |
| 201~300 | 5회 |
| 301+ | 6회 |

교체도 1회로 카운트 (배치=교체, 새 배치=교체)

## 3. DM 배치 플로우

### 3-1. DM에서 "캠프" 입력

1. 거점캠프 설정 여부 확인
2. 현재 라운드 보너스 조건 표시
3. 필드 선택 → 조건 맞는 포켓몬 목록 (점수 미리보기 포함)
4. 배치 확인

```
🏕 캠프 배치 — [크립토라운지] 🌿 숲

🔎 보너스 조건: 이상해씨 (특공 15↑)

추천 포켓몬:
⭐ 이상해씨 [특공:17] → 7점 (보너스+IV)
⭐ 이상해꽃 [특공:12] → 2점 (보너스)
  벌레불 [—] → 1점 (타입만)

[이상해씨 배치]  [이상해꽃 배치]  [전체보기]
```

### 3-2. 채팅방에서 "캠프" 입력

기존과 동일하되, **보너스 조건 + 점수 미리보기** 추가

## 4. 알림 시스템

### 4-1. 라운드 정산 DM

거점캠프 지정 유저에게 정산 결과 DM:

```
🏕 캠프 정산 — [크립토라운지]
━━━━━━━━━━━━━━━
🌿 숲: +3 조각 (4점, 참여자 2명)
🔥 화산: 미배치
💧 호수: +2 조각 (2점, 참여자 3명)
━━━━━━━━━━━━━━━

📋 다음 보너스 조건:
  🌿 숲: 나시 (특방 14↑)
  🔥 화산: 마그마 (공격 16↑)
  💧 호수: 라프라스 (HP 15↑)
```

### 4-2. DM 알림 on/off

- 기본값: ON (거점 설정 시 자동)
- "캠프알림" 명령으로 토글

## 5. 캠프 튜토리얼

기존 튜토리얼(step 1~8)과 별개로, **거점캠프 첫 설정 시 자동 실행**되는 캠프 전용 가이드.

### 5-1. 트리거

- 유저가 처음으로 "거점캠프"를 설정하면 자동 시작
- 또는 DM에서 "캠프가이드" 입력으로 수동 실행
- 이미 완료한 유저는 다시 볼 수 있음 (재실행 가능)

### 5-2. 가이드 플로우 (DM 대화형, 4단계)

**Step 1: 캠프란?**
```
🏕 【캠프 가이드】

캠프는 채팅방에 포켓몬을 배치해서
조각을 모으는 시스템이에요!

📋 흐름:
1️⃣ 거점캠프 설정 (지금 완료!)
2️⃣ 필드에 포켓몬 배치
3️⃣ 3시간마다 자동 정산 → 조각 획득
4️⃣ 조각을 모아 분해결정 교환
5️⃣ 분해결정으로 이로치 전환!

👉 "다음"을 눌러주세요!
```

**Step 2: 배치 방법**
```
🏕 【포켓몬 배치】

캠프에는 여러 필드가 있어요:
🌿 숲 — 풀/벌레/독 타입
🔥 화산 — 불/땅/바위 타입
💧 호수 — 물/얼음/비행 타입
🏙 도시 — 노말/격투/강철 타입
🕳 동굴 — 고스트/악/드래곤 타입
⛩ 신전 — 에스퍼/페어리/전기 타입

각 필드에 맞는 타입의 포켓몬을 배치하세요!

💡 보너스 조건에 맞는 포켓몬을 배치하면
   점수가 훨씬 높아요! (최대 7점)

👉 "다음"을 눌러주세요!
```

**Step 3: 점수와 보너스**
```
🏕 【점수 시스템】

포켓몬 배치 시 점수:
  ✅ 타입만 맞음 → 1점
  ⭐ 보너스 포켓몬 → 2점
  ⭐ 보너스 + 개체값 충족 → 4점
  ⭐ 보너스 + 이로치 → 4점
  🌟 보너스 + 개체값 + 이로치 → 7점

📋 정산 시간 (매일 6회):
  09:00 / 12:00 / 15:00 / 18:00 / 21:00 / 00:00

💡 "거점캠프"를 입력하면 현재 보너스 조건과
   추천 포켓몬을 볼 수 있어요!

👉 "다음"을 눌러주세요!
```

**Step 4: 조각과 보상**
```
🏕 【조각 → 이로치 전환】

정산 시 점수에 따라 조각을 받아요.
조각을 모으면:

💎 분해결정 교환:
  숲 조각 50개 → 분해결정 1개
  (필드마다 필요 조각이 달라요)

✨ 이로치 전환:
  분해결정 5개 → 보유 포켓몬 1마리를 이로치로!

🌈 무지개결정:
  분해결정 10개 → 무지개결정 1개
  무지개결정 1개 → 전설/초전설 이로치 전환!

지금 "거점캠프"를 입력해서
첫 포켓몬을 배치해보세요! 🎉
```

### 5-3. 구현 방식
