> 작성일: 2026-03-07 | 현재: Gen 1-3 (386종), 유저 226명, DAU ~18

## 🧭 핵심 철학 (이 문서의 모든 기획은 여기서 출발한다)

> "이 게임은 게임이지만 텔레그램 내 커뮤니티를 결속시키는 소셜 보조도구의 역할이다.
> 그 본분을 잃어버려선 안 된다."

> "컨텐츠 소비가 빠른 대신, 유저들은 텔레에서 하다보니 텍스트로 ㅊ 치는 거 외에
> 뭘 하기가 많이 어렵다. 이젠 명령어를 더 먹이고 복잡해지는 것보다
> 뉴비를 키우는 방향으로 준비할 거다."

### 우리가 지켜야 할 원칙

1. **명령어를 늘리지 않는다** — 새 기능은 기존 흐름 안에 자연스럽게 녹인다
2. **채팅방의 대화를 방해하지 않는다** — 봇은 보조도구, 주인공은 사람이다
3. **ㅊ 한 번이면 시작이다** — 복잡한 온보딩 없이 즉시 재미를 느끼게 한다
4. **혼자 하는 게임이 아니다** — 모든 기능은 "옆 사람과 함께"를 전제로 설계한다
5. **채널과 상생한다** — 봇이 채널을 키우고, 채널이 봇을 키운다

## 📊 현황 진단: 왜 뉴비가 떠나는가?

### 현재 뉴비 여정 (분석)

```
[1분차] 채팅방에서 야생 포켓몬 출현 → ㅊ 입력 → 포획 성공/실패
    ↓
[1분차] 봇이 DM으로 튜토리얼 시작 (8단계, 10~15분)
    ↓
[???] 유저가 DM을 읽을까? → 대부분 무시하거나 중간에 이탈
    ↓
[???] 돌아와서 ㅊ를 또 칠까? → D+1 리텐션 ~45%
```

### 이탈 포인트 분석

| 구간 | 문제 | 심각도 |
|------|------|--------|
| ㅊ 1회 후 | "재밌었다. 그래서?" — 다음 할 일이 없음 | ★★★★★ |
| 튜토리얼 DM | 텔레 DM은 알림이 밀림. 채팅방에서 벗어나는 순간 맥락 단절 | ★★★★ |
| 2일차 | 돌아올 이유가 없음. 푸시 없음, 리마인드 없음 | ★★★★★ |
| 1주차 | 배틀/교환/미션은 "알아서 찾아야" 하는 구조 | ★★★ |
| 채팅방 분위기 | 기존 유저들 사이에 끼기 어려움. 소속감 없음 | ★★★★ |

### 핵심 인사이트

**뉴비는 "게임을 하러" 온 게 아니다.**
텔레그램 채팅방에서 대화하다가 "어? 포켓몬이 나왔네?" 하고 ㅊ를 친 것뿐이다.
그런데 갑자기 DM으로 튜토리얼이 와서 8단계를 하라고 한다.

**우리가 해야 할 것:**
ㅊ를 친 그 순간의 "어? 재밌는데?" 감정을 유지시키면서,
자연스럽게 다음 ㅊ를 치게 만드는 것.
DM으로 보내는 게 아니라, **채팅방 안에서** 그 경험이 이어져야 한다.

## 1. 7일 뉴비 여정

### 왜 필요한가

현재 튜토리얼은 DM에서 8단계를 한 번에 소화시킨다.
그런데 텔레 유저는 DM을 잘 안 본다. 본다 해도 한꺼번에 10분 투자하기 싫어한다.

**해결책: 튜토리얼을 DM 한 방이 아니라, 7일에 걸친 "자동 여정"으로 바꾼다.**
매일 하나씩 작은 미션을 주고, 그때그때 보상을 준다.
유저 입장에선 "아, 오늘 할 거 있네" 하고 1분이면 끝나는 것.

### 상세 메커니즘

**Day 1: 첫 포획**
- 트리거: 첫 `/ㅊ` 입력
- 보상: 즉시 하이퍼볼 3개 + 채팅방 인라인 축하 메시지
  - "🎉 {이름}님이 첫 포켓몬을 잡았습니다! 축하해요!"
  - 다른 유저들이 보고 반응할 수 있는 **공개 메시지** (DM 아님!)
- 자동 DM (간단하게 2줄):
  - "첫 포켓몬을 잡았네요! 내일 다시 오면 특별한 선물이 있어요 🎁"
- 효과: 첫날은 ㅊ만 치면 됨. 부담 없음. "내일 뭐 주는데?" 궁금증 유발

**Day 2: 돌아왔다! (D+1 리텐션 핵심)**
- 트리거: 2일차에 아무 채팅방에서 ㅊ 입력 (자동 감지)
- 채팅방 인라인: "🔥 {이름}님이 2일 연속 출석! 마스터볼 1개 획득!"
- 보상: 마스터볼 1개 + BP 100
- 자동 DM 팁: "마스터볼은 아무 포켓몬이나 100% 확률로 잡아요. /ㅁ 으로 사용!"
- 효과: D+1 리텐션이 가장 중요. "돌아오면 마스터볼" 이 한 마디로 유저가 2일차에 옴

**Day 3: 밥주기 (육성 경험)**
- 트리거: 3일차 ㅊ 입력 시
- 채팅방 인라인: "📋 {이름}님, 3일차 미션! → 내 포켓몬에게 밥 주기 (DM에서 /밥 1)"
- 보상: 하이퍼볼 2개 + BP 50
- 자동 DM: "밥을 주면 친밀도가 올라가고, 포켓몬이 더 강해져요!"
- 효과: DM 기능을 처음으로 자연스럽게 유도 (밥 한번이면 끝)

**Day 4: 도감 구경 (수집 동기 부여)**
- 트리거: 4일차 ㅊ 입력 시
- 채팅방 인라인: "📋 {이름}님, 4일차 미션! → /도감 으로 내 수집 현황 확인!"
- DM에서 /도감 입력 시 자동 완료
- 보상: BP 100 + 강스권 1장
- 효과: "아 나 아직 이것밖에 없구나" → 수집 욕구 자극

**Day 5: 첫 배틀 (PvP 경험)**
- 트리거: 5일차 ㅊ 입력 시
- 채팅방 인라인: "📋 {이름}님, 5일차 미션! → 아무에게나 /배틀 도전!"
  - 또는 자동 매칭: "NPC 트레이너 영희" 와 AI 대전 (포켓몬 3마리 약한 NPC)
- 보상: 마스터볼 1개 + BP 200
- 효과: 배틀 시스템 체험. NPC 대전이면 부담 없이 "이기는 경험" 제공

**Day 6: 교환 체험 (소셜 연결)**
- 트리거: 6일차 ㅊ 입력 시
- 채팅방 인라인: "📋 {이름}님, 6일차 미션! → /거래소 에서 포켓몬 구경하기"
- /거래소 입력만 하면 자동 완료 (구매 안 해도 됨)
- 보상: BP 200
- 효과: 거래소 존재를 인지 + "이런 포켓몬들이 있구나" 동기 부여

**Day 7: 졸업 + 칭호 수여**
- 트리거: 7일차 ㅊ 입력 시
- 채팅방 인라인 (축하 이벤트):
  ```
  🎓 축하합니다! {이름}님이 7일 뉴비 여정을 완주했습니다!

  🏅 칭호 해금: 「🌟 신예 트레이너」
  🎁 졸업 보상: 마스터볼 3개 + BP 500

  {이름}님, 이제 진정한 트레이너입니다! 🎉
  ```
- 효과: 7일간 보상 합계가 상당하므로 완주 동기 + 졸업 후 자연스럽게 일상 유저로 전환

### 7일 보상 총합

| 일차 | 행동 | 보상 |
|------|------|------|
| 1일 | ㅊ 1회 | 하이퍼볼 3 |
| 2일 | ㅊ 1회 | 마스터볼 1 + BP 100 |
| 3일 | 밥 1회 | 하이퍼볼 2 + BP 50 |
| 4일 | /도감 1회 | BP 100 + 강스권 1 |
| 5일 | 배틀 1회 | 마스터볼 1 + BP 200 |
| 6일 | /거래소 1회 | BP 200 |
| 7일 | ㅊ 1회 (졸업) | 마스터볼 3 + BP 500 |
| **합계** | | **마스터볼 5 + 하이퍼볼 5 + BP 1150 + 강스권 1 + 칭호** |

### 핵심 설계 원칙

- **채팅방에서 인라인으로 안내** → DM만 보내면 유저가 안 봄
- **하루 1가지만** → "오늘 할 거" 가 명확함
- **1분이면 끝남** → 부담 없음
- **공개 축하** → 다른 유저들이 "오 축하!" 반응 → 소셜 결속
- **연속 접속 보상 체감** → 2일차 마스터볼, 7일차 마스터볼 3개

### 기존 튜토리얼과의 관계

- 기존 DM 튜토리얼은 **선택사항**으로 유지 (/튜토로 언제든 가능)
- 7일 여정은 **자동으로** 진행됨 (유저가 ㅊ만 치면 됨)
- 7일 여정 완주 = 기존 튜토리얼 졸업과 동일한 칭호 + 보상 체계

### 우리가 얻는 것

- **D+1 리텐션 45% → 목표 65%+**: "내일 마스터볼" 한 마디가 핵심
- **D+7 리텐션 측정 가능**: 현재는 D+1만 추적. 7일 여정으로 주간 리텐션 추적
- **기능 인지도 향상**: 배틀/교환/도감을 "해봤으니까" 나중에 자발적으로 사용
- **커뮤니티 일체감**: 공개 축하 메시지로 채팅방 분위기 활성화

### 구현 난이도: ★★★☆☆
- DB: `user_journey` 테이블 (user_id, day, completed_at)
- 로직: ㅊ 핸들러에서 journey day 체크 → 인라인 메시지 + 보상 지급
- 기존 코드 수정 최소화 (group.py의 catch 핸들러에 hook 추가)

## 2. 채팅방 레벨 시스템

### 왜 필요한가

> "대형채널에서 봇을 추가하면 그 채팅방만의 혜택을 따로 줄 수 있을지"

현재 봇을 추가한 채팅방은 "포켓몬이 나오는 곳" 그 이상도 이하도 아니다.
채널 운영자 입장에서 봇을 넣을 인센티브가 부족하다.

**해결책: 채팅방 자체에 레벨과 혜택을 부여한다.**
활동이 활발한 채팅방일수록 더 좋은 포켓몬이 나오고, 특별 이벤트가 열린다.
채널 운영자는 "우리 방에서 포켓몬하면 좋아요!"라고 자랑할 수 있다.

### 상세 메커니즘

**채팅방 경험치 (CXP)**
- 포켓몬 포획 성공: +1 CXP
- 채팅방 내 배틀: +2 CXP
- 교환 완료: +1 CXP
- 일일 최대: 50 CXP (봇 어뷰징 방지)

**채팅방 레벨 & 혜택**

| 레벨 | 필요 CXP | 혜택 |
|------|----------|------|
| Lv.1 | 0 | 기본 (일 2~3회 스폰) |
| Lv.2 | 100 | 스폰 +1회/일 |
| Lv.3 | 300 | 에픽 확률 +10% |
| Lv.4 | 600 | 스폰 +2회/일 + 전설 확률 +5% |
| Lv.5 | 1000 | "명예의 전당" 해금 (이 방 최초 포획 기록 표시) |
| Lv.6 | 1500 | 주간 보너스 스폰 (토요일 특별 포켓몬 1회) |
| Lv.7 | 2000 | 이로치 확률 +3% 상시 |
| Lv.8 | 3000 | 채팅방 전용 칭호 ("🏠 [방이름] 주민") |
| Lv.9 | 5000 | 레이드 보스 출현 자격 |
| Lv.10 | 8000 | 채팅방 리더보드 + 월간 MVP 시상 |

**채팅방 프로필 카드**
- `/방정보` 명령어로 조회 (기존 명령어 체계 내)
```
📊 [채팅방 이름] 프로필
━━━━━━━━━━━━━━
🏅 레벨: Lv.5 (CXP 1,234/1,500)
👥 활동 트레이너: 23명
🎯 총 포획수: 1,847마리
✨ 이로치 출현: 12회
🏆 명예의 전당: 3종 최초포획
━━━━━━━━━━━━━━
다음 레벨까지: 266 CXP
```

### 채널 운영자 인센티브

- **레벨이 높은 채팅방 = 포켓몬이 잘 나오는 방** → 트레이너들이 찾아옴
- **전용 칭호** → "나 이 방 주민이야" 소속감
- **명예의 전당** → 채팅방 고유의 역사와 기록 → 자부심
- **운영자 대시보드**: tgpoke.com에서 내 채팅방 통계 확인 가능

### 우리가 얻는 것

- **채널 바이럴**: 운영자가 "우리 방 Lv.5 됐다!" 자랑 → 신규 유입
- **봇 추가 유인**: 채널 운영자가 자발적으로 봇 추가 (혜택이 있으니까)
- **활동량 증가**: 레벨업을 위해 멤버들이 더 적극적으로 포획
- **다중 채팅방 활동**: 여러 방에서 활동 → 전체 DAU 증가

### 구현 난이도: ★★☆☆☆
- DB: `chat_levels` 테이블 (chat_id, cxp, level)
- 로직: 포획/배틀 시 CXP 적립 → 레벨업 체크 → 혜택 적용
- 혜택은 기존 스폰 로직의 확률 파라미터만 조정

## 3. 주간 시즌 배틀

### 왜 필요한가

현재 배틀은 누적 BP 기준 랭킹뿐. 한번 벌어진 격차는 좁혀지지 않는다.
**1주 단위로 리셋하고, 매주 다른 규칙**을 적용하면 신선함이 유지된다.

### 상세 메커니즘

**시즌 구조**
- 기간: 1주 (월요일 00시 ~ 일요일 23:59)
- 리셋: 매주 월요일 RP(랭크포인트) 소프트리셋 (이전 RP × 0.3)
- 티어: 브론즈 → 실버 → 골드 → 플래티넘 → 다이아 → 마스터

**주간 특수 규칙 (핵심 차별점)**

매주 월요일 자동 공지로 이번 주 규칙 발표:

| 주차 | 규칙 이름 | 설명 |
|------|----------|------|
| 1주차 | 🚫 초전설 금지 | 초전설(뮤츠/루기아/칠색조/레쿠쟈) 팀 등록 불가 |
| 2주차 | 🌿 풀속성 위크 | 풀 타입 포켓몬 필수 1마리 이상 편성 |
| 3주차 | ⭐ 레어 필수 | 레어 등급 포켓몬 최소 2마리 편성 |
| 4주차 | 🎲 랜덤 3마리 | 팀 6마리 중 3마리가 랜덤 배정 |
| 5주차 | 🔥 단일 속성 | 팀 전원 같은 타입으로 편성 |
| 6주차 | 👶 1진화 전용 | 3단계 진화 포켓몬 사용 금지 |
| 7주차 | 🏆 제한 없음 | 풀 오픈 (최강전) |
| 8주차 | 🎯 IV 역전 | IV 합계 낮은 포켓몬이 유리 (저IV에 보정치) |

규칙 풀은 15~20개 준비 → 매주 랜덤 배정 → 같은 규칙이 연속으로 안 나오게

**이게 재밌는 이유:**
- "이번 주는 풀속성 위크래" → "나 이상해꽃 있는데!" 자연스러운 대화 유발
- 매주 팀을 바꿔야 함 → 다양한 포켓몬에 가치 부여 → 수집 동기 강화
- 초전설 없는 주 → 뉴비도 이길 수 있음 → 공정성

**보상**
| 티어 | 주간 보상 |
|------|----------|
| 마스터 | 마스터볼 3 + 주간 전용 칭호 |
| 다이아 | 마스터볼 2 |
| 플래티넘 | 마스터볼 1 + BP 300 |
| 골드 | BP 500 |
| 실버 | BP 200 |
| 브론즈 | BP 100 |

### 명령어 (최소한)
- 기존 `/배틀` 에 랭크 옵션 추가 (별도 명령어 아님)
  - `/배틀` → "일반 배틀 / 랭크 배틀" 버튼 선택
- `/시즌` → 이번 주 규칙 + 내 RP + 티어

### 우리가 얻는 것

- **주간 리텐션**: "이번 주 규칙 뭔데?" 궁금증으로 매주 접속
- **밸런스 자동 해결**: 초전설 금지 주에는 뉴비도 선전 가능
- **팀 다양성**: 매주 팀을 바꿔야 하니 "이 포켓몬도 키워야겠다"
- **커뮤니티 대화**: 주간 규칙이 자연스러운 채팅 소재

### 구현 난이도: ★★★☆☆
- DB: `season_records` (user_id, season_id, rp, tier, wins, losses)
- 규칙 검증: 팀 등록 시 이번 주 규칙에 맞는지 체크
- 스케줄러: 매주 월요일 리셋 + 보상 분배

## 4. 합성 시스템

### 왜 필요한가

IV가 좋은 포켓몬을 원하지만 방법이 "운 좋게 잡기"밖에 없다.
중복 포켓몬은 쌓이기만 하고, 방생하면 하이퍼볼 1개.
→ 중복 포켓몬 2마리를 합쳐서 더 좋은 IV의 포켓몬을 만드는 시스템.

### 상세 메커니즘

**동종 합성**
- 같은 종 2마리 → 1세대 형태 1마리 (친밀도 0부터)
- IV: 부모 중 높은 값 70% 상속, 30% 랜덤
- 비용: 100 BP
- 1일 5회 제한

**이종 합성**
- 같은 등급 다른 종 2마리 → 같은 등급 랜덤 1마리
- IV: 완전 랜덤
- 비용: 200 BP
- 도감에 없는 종 우선 배정

**제한:** 전설/초전설 합성 불가, 팀 등록/거래소 등록 중 불가

### 우리가 얻는 것

- **BP 소비**: 현재 BP 축적만 되는 구조 해결
- **중복 포켓몬에 가치**: "재료"로서의 쓸모
- **엔드게임**: IV S등급 목표 → 장기 플레이 유지
- **거래소 활성화**: 합성 재료로 거래 수요 증가

### 구현 난이도: ★★★☆☆

## 5. 주간 미션 & 업적

### 왜 필요한가

일일 미션 4개는 좋지만 "매일 같은 것"이라 피로감이 있다.
주간 미션은 좀 더 여유롭게 + 다양한 행동을 유도한다.

### 상세 메커니즘

**주간 미션 (월요일 초기화, 3개 랜덤)**

| 미션 | 목표 | 보상 |
|------|------|------|
| 포획 | 20마리 잡기 | BP 200 + 하이퍼볼 3 |
| 배틀 | 5승 | BP 300 + 마스터볼 1 |
| 친밀도 | MAX 3마리 | BP 150 + 하이퍼볼 5 |
| 진화 | 3회 | BP 200 + 강스권 1 |
| 도감 | 신규 5종 | BP 200 + 마스터볼 1 |

주간 올클리어: 마스터볼 2 + BP 500

**단계별 업적 (영구 누적)**

총 7개 카테고리 × 3단계. 3단계 달성 시 전용 칭호 해금.
- 포획왕 (100/500/2000마리)
- 배틀광 (50/200/1000승)
- 육성가 (MAX친밀 10/50/150)
- 수집광 (도감 100/250/386종)
- 부자 (BP 누적 10k/50k/200k)
- 이로치 (5/20/50마리)
- 교환왕 (10/50/200회)

### 우리가 얻는 것

- **중장기 목표**: "다음 업적까지 얼마 남았지?"
- **다양한 활동 유도**: 포획만 하던 유저 → 배틀/교환도 시도
- **칭호 수집 동기**: 칭호 자체가 보상

### 구현 난이도: ★★☆☆☆
- 기존 미션 시스템 확장 (mission_service.py에 weekly 모드 추가)

## 6. 채널 네트워크 & 크로스 프로모션

### 왜 필요한가

> "봇을 통해 새로운 채팅방이 ~~~하니까 거기 가보세요 하면서
> 링크 주고 홍보할 수도 있을 거고"

현재 봇이 추가된 채팅방들은 서로의 존재를 모른다.
채팅방끼리 연결해주면 유저 풀이 커지고, 각 채팅방도 멤버가 늘어나는 상생 구조.

### 상세 메커니즘

**채팅방 디렉토리**
- 봇이 추가된 채팅방 중 **운영자가 동의한 방**만 디렉토리에 등록
- `/채팅방목록` → 활발한 채팅방 리스트 (레벨순)
```
🏠 활발한 포켓몬 채팅방
━━━━━━━━━━━━━━━━━
1. 🏅 Lv.7 크립토 라운지 (45명) — "이로치 성지"
2. 🏅 Lv.5 텔레 포켓몬 (32명) — "뉴비 환영!"
3. 🏅 Lv.4 코인방 (28명) — "배틀 활발"
━━━━━━━━━━━━━━━━━
→ 방 이름 클릭 시 초대 링크
```

**크로스 프로모션 시스템**
- 채팅방 Lv.3+ 운영자가 **홍보 메시지 등록** 가능
- 다른 채팅방에서 포켓몬 포획 시 5% 확률로 한 줄 추가:
  ```
  📢 [크립토 라운지] 에서도 포켓몬을 잡을 수 있어요! (Lv.7 보너스 적용 중)
  ```
- 클릭하면 해당 채팅방 초대 링크
- 하루 최대 3회만 표시 (스팸 방지)

**채팅방 이적 보너스**
- 유저가 새로운 채팅방에서 처음 ㅊ를 치면:
  - "🆕 {이름}님이 새로운 채팅방에서 포켓몬 여행을 시작합니다!" (인라인)
  - 보상: BP 50 (채팅방당 1회)
- 신규 방 운영자에게도 CXP +5 보너스

### 우리가 얻는 것

- **유저 풀 확대**: 채팅방 A 유저가 B로 이동 → 전체 DAU 증가
- **채널 운영자 만족**: "우리 방 홍보해주네" → 봇 유지 동기
- **자연스러운 성장**: 광고 없이 봇이 추가된 방끼리 서로 알리는 구조
- **커뮤니티 네트워크**: "포켓몬하는 텔레 방" 생태계 형성

### 구현 난이도: ★★★☆☆
- DB: `chat_directory` 테이블 (chat_id, name, invite_link, description, opt_in)
- 운영자가 `/방등록` 으로 opt-in → 관리자 승인 후 디렉토리 등재

## 7. 행운의 뽑기

### 왜 필요한가

매일 접속할 이유가 "포켓몬이 나올 때까지 기다리기"밖에 없다.
출석체크(하이퍼볼 1개)도 있지만 체감이 약하다.
**1일 1회 무료 뽑기**로 "오늘 뭐 나올까?" 기대감 제공.

### 상세 메커니즘

- `/뽑기` — 1일 1회 무료 (자정 초기화)
- 결과 (확률):

| 등급 | 보상 | 확률 |
|------|------|------|
| 💎 대박 | 마스터볼 1개 | 1% |
| ✨ 행운 | 이로치 부스트 1시간 | 2% |
| 🎉 좋음 | 하이퍼볼 3개 | 12% |
| 👍 괜찮음 | BP 50~100 | 35% |
| 😊 기본 | BP 10~30 + 하이퍼볼 1 | 50% |

- 채팅방에서 치면 인라인 결과 표시 (다른 유저도 봄)
  - "🎰 {이름}님의 오늘의 운세... 💎 대박! 마스터볼 1개!!"
  - → 다른 유저: "와 부럽다 ㅋㅋ" → 소셜 인터랙션

### 우리가 얻는 것

- **일일 접속률 향상**: "오늘 뽑기 했어?" 가 대화 소재
- **소셜 인터랙션**: 결과가 공개되니 반응이 옴
- **비용 0**: 평균 보상은 BP 25~30 수준으로 경제 영향 미미

### 구현 난이도: ★☆☆☆☆
- 가장 간단한 구현. 랜덤 함수 + 보상 지급 + 인라인 메시지
