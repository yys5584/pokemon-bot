성격 시스템 설계

2026-03-30 확정 · 카드 등급 폐기 → 성격 16종 통합

1. 핵심 목적

  • 잡는 행위 자체에 재미/의미 부여 — 도감 완성 후에도 ㅊ 칠 이유
  • 채팅방 소셜 이벤트 — 좋은 성격 뜨면 자랑, 반응, 구경
  • 수집욕 자극 — TCG 레퍼런스, 카드 색 = 직관적 희귀도
  • 배틀 몰라도 즉시 이해 가능

2. 성격 체계

성격 = 시스템의 전부. 별도 "등급" 개념 없음. 성격 이름 안에 티어(강도)와 유형(방향)이 내재.

16종 (4티어 × 4유형)

티어공격형 (ATK/SPA)방어형 (DEF/SPDEF)스피드형 (SPD)밸런스형
T1 (45%)겁쟁이유리몸느긋함변덕
T2 (30%)용감강철재빠름성실
T3 (15%)사나움억센질풍평정심
T4 (10%)귀신불굴신속전설
  • 티어 = IV 상한선 최적화 강도 (높을수록 좋음)
  • 유형 = IV 상한선을 몰아주는 방향

"귀신 리자몽" = 최상위 공격형 → 금색 카드 → "미쳤다"

"겁쟁이 리자몽" = 하위 공격형 → 회색 카드 → 흔함

3. IV 상한선 시스템

기존 IV 등급 (S/A/B/C/D) — 유지

실제 롤된 총합 / 186 기준. 예: 140/186 표시

성격이 결정하는 것

  • 유형: 어떤 스탯에 상한선을 몰아줄지 (방향)
  • 티어: 얼마나 몰아줄지 (강도)
  • 스탯당 범위: 15 ~ 46
  • 상한선 총합: 186 (고정)

티어별 최적화 강도

티어최적화 %설명
T10%완전 랜덤 배분
T233%약간 몰아줌
T366%꽤 몰아줌
T4100%유형 방향으로 완전 최적화

공식: ceiling = random × (1 - opt%) + optimal × opt%

유형별 IV 상한선 배분

  • 공격형 (귀신/사나움/용감/겁쟁이): ATK, SPA에 상한선 높게
  • 방어형 (불굴/억센/강철/유리몸): DEF, SPDEF에 높게
  • 스피드형 (신속/질풍/재빠름/느긋함): SPD에 높게
  • 밸런스형 (전설/평정심/성실/변덕): 전체 고르게

예시: 귀신(T4 공격형) 리자몽

ATK 상한 28, SPA 상한 44, SPD 상한 35, DEF 상한 20, SPDEF 상한 20, HP 상한 39 (합 186)

겁쟁이(T1 공격형)면 같은 공격형이지만 거의 랜덤 배분

4. 배틀 영향

IV → 배틀 배수 공식 (현행 유지)

배수 = 0.85 + (IV / 31) × 0.30

IV 15 = 1.00x  |  IV 31 = 1.15x (현재 만렙)  |  IV 46 = 1.30x

  • T4 일반(127) < T1 전설(154) — 티어 역전은 에픽부터만

6. 포켓몬 유형 표시

베이스스탯 기반 자동 판정. 도감 + 스폰 카드 하단에 표시. 성격 부여에 영향 없음.

유형판정 기준
공격형ATK+SPA 최고
방어형DEF+SPDEF 최고
스피드형SPD 최고
밸런스형차이 적음

7. 레거시 포켓몬

  • 마이그레이션 스크립트로 전체 랜덤 부여 (45/30/15/10%)
  • 기존 IV는 변경 없음, 성격만 신규 부여
