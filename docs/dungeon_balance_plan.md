## 현재 문제 (실제 데이터 198런 기준, 2026-03-19)
- 에픽 avg 20.6F (max 43F) > 레어 18.0F > 전설 16.9F > 초전설 15.0F (max 29F)
- **등급 높을수록 낮음** — 버프 빈도(DUNGEON_BUFF_FREQUENCY)가 코스트에 반비례
- 게을킹(에픽) 독주: Top 20 중 11개

## 변경 사항 (미구현, 예비 계획)

### 1. IV스톤 포켓몬당 3회 제한
- `user_pokemon`에 `iv_stone_used INT DEFAULT 0` 컬럼 추가
- `queries.apply_iv_stone()`: iv_stone_used < 3 체크 + 사용 시 +1
- `dm_gacha.py` UI에 남은 횟수 표시

### 2. 버프 빈도 조정
- 현재: `{1:1, 2:1, 4:2, 5:3, 6:4}` — 초전설 4층마다
- 변경안: `{1:1, 2:1, 4:2, 5:2, 6:2}` — 전설/초전설 2층마다
- 또는 전 등급 통일 후 적 스케일링으로 밸런스

### 3. 적 스케일링 상향
- 40층+ 급격히 어렵게 (현재 너무 완만)
- 현재: `floor 41+ = 2.50 + (floor-40) × 0.08`
- 변경안: 40+ 구간 기울기 대폭 상향

### 4. 50층 보스
- 50F 보스는 초전설급 스탯이 아니면 돌파 불가능한 수준
- 초전설 S급 + IV올맥 기준으로 역산
