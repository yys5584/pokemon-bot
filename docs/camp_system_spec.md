## 1. 왜 필요한가

### 현재 구조

```
허슬형: ㅊ → 팀구성 → 배틀 → 토너먼트 → 랭크전 (엔드컨텐츠 ✅)
라이트형: ㅊ → 도감작 → ... → 현타 ("이걸 왜 모으지?") (엔드컨텐츠 ❌)
```

### 핵심 문제

라이트 유저는 포켓몬을 모으지만, 모은 포켓몬으로 **할 수 있는 게 배틀밖에 없다.**
배틀은 고인물의 영역이고, 덱짜기는 진입장벽이 높다.
결국 도감작만 하다가 "이걸 왜 하지?" 현타가 오고 이탈한다.

### 해결 방향

**모은 포켓몬이 살아 움직이는 공간을 만든다.**
내 포켓몬이 채팅방에서 뭔가를 하고, 그게 대화가 되고, 보상이 된다.
캠프가 있는 채팅방 = "도시" → 유저 유입 + 바이럴 엔진.

## 2. 디자인 원칙

### 원칙 1: 버튼 하나 (네코 아츠메)
**"포켓몬 배치"가 유일한 능동적 행동.** 나머지는 전부 자동.

### 원칙 2: 같은 문제, 같은 대화 (워들)
라운드마다 보너스 포켓몬이 전 유저에게 동일 → 대화 유도.

### 원칙 3: 벌주지 않기 (동숲)
안 하면 조각을 못 모을 뿐, 기존 것이 사라지지 않음. **출석 보상 X, 압박 없음.**

### 원칙 4: 흔한 게 귀해지는 순간 (스타듀밸리)
라운드 보너스 포켓몬에 흔한 포켓몬이 지정되면 갑자기 가치 생김.

### 원칙 5: 채팅방을 더럽히지 않기
하루 캠프 메시지 최대 6~7개. 자동 삭제. 짧고 깔끔.

## 6. 라운드 구조

### 소식 스케줄 (3시간 간격, 하루 6회)

```
09:00 / 12:00 / 15:00 / 18:00 / 21:00 / 00:00
```

> 대회 접수 알림은 09:10 (캠프와 10분 간격)

### 1라운드 = 3시간

```
09:00~09:30  접수 타임 (30분)
 ├ 자유슬롯: 선착순 배치
 └ 승인슬롯: 소유자에게 알림 → 30분 내 미응답 시 자동 승인

09:30~12:00  파밍 타임 (2시간 30분)
 └ 배치된 포켓몬 조각 생산

12:00        정산 + 다음 라운드 접수 시작
```

### 라운드 보너스 포켓몬

매 라운드마다 각 필드에 **보너스 포켓몬 + 개체값 조건** 랜덤 지정:

```
[09:00 라운드]
🌿 숲: 이상해꽃 (특공 15↑)
🔥 화산: 리자몽 (공격 14↑)
💧 호수: 갸라도스 (방어 13↑)
```

- 해당 포켓몬 배치 시 점수 2배
- 없으면 타입 맞는 아무거나 OK
- 라운드마다 변경 → 다양한 포켓몬이 가치 생김

## 7. 점수 & 생산 시스템

### 배치 포켓몬 점수

| 조건 | 점수 |
|------|------|
| 타입만 맞음 | 1점 |
| **보너스 포켓몬** | **2점** |
| **보너스 + 개체값 충족** | **4점** |
| **보너스 + 이로치** | **4점** |
| **보너스 + 개체값 + 이로치** | **7점** |

### 생산 공식

```
필드 생산(소식당) = min(필드 총점, 캠프캡) → 배치자 전원 엔빵
```

### 캠프 레벨별 캡

| Lv | 캡 (조각/소식/필드) |
|----|-------------------|
| 1 | 3 |
| 2 | 3 |
| 3 | 4 |
| 4 | 4 |
| 5 | 5 |
| 6 | 5 |
| 7 | 6 |
| 8 | 6 |
| 9 | 7 |
| 10 | 8 |

### 조각 = 필드 귀속

- 숲 필드 → **숲 조각** 획득
- 숲 조각으로 풀/벌레/독 타입 이로치 전환 가능
- 리자몽(불) 이로치 → **화산 조각** 필요
- 듀얼타입은 **둘 중 하나** 선택 가능

### 수입 계산 (4점 평균, 필드 꽉참 기준)

## 8. 배치 제한

### 인당 일일 배치 횟수 (도감 기반)

| 도감 수 | 일일 배치 횟수 |
|---------|-------------|
| 0~50 | 2회 |
| 51~100 | 3회 |
| 101~200 | 4회 |
| 201~300 | 5회 |
| 301+ | 6회 |

- "배치 횟수" = 하루 동안 슬롯에 포켓몬을 넣는 행위 횟수
- 한번 넣으면 로테이션으로 빠질 때까지 유지
- 빠진 후 다시 넣으려면 횟수 소모
- 자정 리셋
- FCFS 어뷰징 방지 + 도감작 보상

## 15. UX 요약

### 유저가 배워야 할 것

```
1. 소식 뜨면 접수 타임에 배치한다 (30분)
2. 보너스 포켓몬 맞추면 점수 높다
3. 조각 모으면 이로치 된다
```

### 하루 일과 (적극적 유저)

```
09:00 — 소식 + 보너스 포켓몬 확인 → 배치 (30초)
12:00 — 소식, 필요시 재배치
15:00 — 소식
18:00 — 소식
21:00 — 소식 (대회와 겹치지 않게 대회는 09:10)
00:00 — 마지막 소식
```

총 능동 행동: 배치 1~2회 (30초씩). 나머지 패시브.

### 소극적 유저

배치 안 바꾸면 이전 라운드 유지. 조각은 계속 들어옴. 벌칙 없음.
