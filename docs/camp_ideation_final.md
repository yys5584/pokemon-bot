## 🔥 Tier S — 반드시 해야 할 것 (캠프의 뼈대)

### 1. 캠프 날씨 시스템 🅰️
⭐⭐⭐⭐⭐ 실현 | 🔥🔥🔥🔥 참여 | 💎💎💎 독창

**무엇:** 매일 캠프에 랜덤 날씨가 적용. ☀️맑음(풀숲 2배), 🌧️비(호수 2배), ⛈️폭풍(전기탑 3배, 실패 확률 10%), 🌫️안개(그림자숲 2배), ❄️눈(모든 구역 +1 고정).

**왜:** "오늘 날씨 뭐야?"가 코인방의 "오늘 시장 어때?"와 같은 인사말이 된다. 워들 효과 — 같은 날씨를 보고 같은 전략을 고민하는 공유된 경험.

**구현:** 임무 생성 시 `random.choice(WEATHERS)` + 해당 구역 `frag_mult` 적용. 아침 공지에 날씨 이모지 1줄 추가. **코드 10줄 수준.**

```
🏕 오늘의 캠프 (3/14) 🌧️ 비
━━━━━━━━━━━━━
💧 호수: 갸라도스 (스피드 15+) ← 날씨 보너스 ×2!
🌿 풀숲: 이상해꽃 (힘 20+)
🔥 모닥불: 아무 불타입
```

### 2. 캠프 레벨 랜드마크 축하 + 칭호 🅰️
⭐⭐⭐⭐⭐ 실현 | 🔥🔥🔥🔥 참여 | 💎💎 독창

**무엇:** 캠프 레벨 5/10/15/20 달성 시 특별 축하 메시지 + 그룹 전원 한정 칭호 + 슬롯 보너스.

| 레벨 | 칭호 | 보너스 |
|------|------|--------|
| 5 | 🏚️ 통나무집 주민 | 전원 조각 +1/일 |
| 10 | 🏛️ 광장 개척자 | 배치 슬롯 +1 |
| 15 | 📚 도서관 학자 | 이벤트 조각 1.5배 |
| 20 | ✨ 포켓몬 왕국 시민 | 이로치 전환 비용 -10% |

**왜:** "우리 방 광장 됐다!" = 공동 달성감. 칭호가 "이 방 소속"이라는 정체성을 만든다.

**구현:** `_check_camp_level_up()`에 마일스톤 분기 + `grant_title()` 호출. 기존 칭호 시스템 재활용.

### 3. 포켓몬 일기 (DM 마이크로 서사) 🅰️
⭐⭐⭐⭐⭐ 실현 | 🔥🔥🔥🔥🔥 참여 | 💎💎💎💎 독창

**무엇:** DM에서 `내캠프` 조회 시, 오늘 배치된 포켓몬들의 "일기"가 표시된다.

```
📖 오늘의 캠프 일기
━━━━━━━━━━━━━
🌿 이상해꽃: "오늘 풀숲에서 나비를 쫓았다. 씨앗 하나가 싹텄다."
💧 꼬부기: "호수에서 조개 3개를 주웠다. 파이리한테 하나 줬다."
🔥 파이리: "꼬부기가 준 조개를 모닥불에 구워봤는데... 맛있었다."
━━━━━━━━━━━━━
🧩 오늘 획득 조각: 🟢×2 🔵×3 🔴×1
```

**왜:** Character.AI의 "기억" 메커니즘. 유저가 매일 DM을 확인하는 습관을 만든다. 텍스트 게임의 핵심 — "짧은 서사가 상상을 자극한다." 채팅방을 안 더럽히면서도 감정적 유대를 만드는 유일한 방법.

**구현:** 일기 템플릿 30개+ (구역별 × 포켓몬 타입별). `내캠프` 핸들러에서 오늘의 소식 로그 기반 일기 조합. 포켓몬끼리 상호참조하는 크로스 일기가 핵심 차별점.

**무엇:** 스타듀밸리 커뮤니티센터처럼, 매주 채팅방 전체가 협력하여 달성하는 그룹 목표.

```
📦 이번 주 캠프 번들: 🌊 물의 축제
━━━━━━━━━━━━━━━━
☐ 물타입 5마리 배치 (현재 3/5)
☐ 파랑 조각 30개 수집 (현재 18/30)

━━━━━━━━━━━━━━━━

**왜:** "꼬부기 있는 사람?" "나 물타입 3마리 넣었어!" — 자연스러운 역할 분담과 대화 발생. Dragon City의 얼라이언스 체스트가 검증한 패턴.

**구현:** `camp_weekly_bundles` 테이블 + 월요일 자정 생성 + 진행도 트래킹. 번들 조건 풀 20개+ 준비.

## 💫 Tier A — 강력 추천 (게임의 깊이를 만드는 것)

### 6. 포켓몬 파트너 시스템 (AI 컴패니언) 🅱️
⭐⭐⭐ 실현 | 🔥🔥🔥🔥🔥 참여 | 💎💎💎💎💎 독창

**무엇:** 유저가 1마리를 "파트너"로 지정. 파트너 포켓몬에게:
1. **이름 부여** — "물방울이" (꼬부기)
2. **성격 선택** — 장난꾸러기 / 수줍음 / 활발함 / 느긋함
3. **DM 인터랙션** — 파트너가 가끔 유저에게 DM을 보냄

```
💬 물방울이(꼬부기)가 메시지를 보냈어요!

"주인~ 오늘 호수에서 큰 물고기 봤어!
잡으려고 했는데 너무 빨라서 놓쳤어... 😢
내일은 꼭 잡을 거야! 💪"
```

- 장난꾸러기 성격이면 "ㅋㅋ 주인 오늘 배틀에서 졌지~? 나라면 이겼을텐데!"
- 수줍음 성격이면 "...오늘도 호수가 예뻤어요."
- 성격별 말투 템플릿 20개+ × 4성격 = 80개+

**Phase 2:** 템플릿 기반 DM (하루 1회, 오전 10시~오후 8시 랜덤)
**Phase 4+:** LLM 연동으로 자유 대화 가능 (비용 관리 필요)

**왜:** 유저의 가장 큰 요구 — "잡은 포켓몬에 영혼을 불어넣고 싶다." Character.AI가 증명한 감정적 유대. Tamagotchi의 핵심 — "나를 필요로 하는 존재." 텔레그램 DM이라는 1:1 공간이 친밀함을 극대화한다.

**구현:**
- `user_partners` 테이블 (user_id, pokemon_instance_id, nickname, personality)
- 성격별 DM 템플릿 세트
- 스케줄러: 하루 1회 파트너 DM 전송 (접속 중인 유저만)
- DM에서 간단한 반응 버튼: [쓰다듬기 🤚] [간식주기 🍎] → 친밀도 +

### 7. 포켓몬 짝꿍 (관계 시스템) 🅱️
⭐⭐⭐⭐ 실현 | 🔥🔥🔥🔥 참여 | 💎💎💎💎 독창

**무엇:** 같은 구역에 7일+ 함께 배치된 포켓몬끼리 "짝꿍" 관계 형성. 짝꿍 효과:
- 함께 등장하는 특별 소식 ("꼬부기와 파이리가 호수에서 물싸움 중! 💦🔥")
- 같이 배치 시 조각 +1 보너스
- 짝꿍 해제 시 "이별 소식" ("꼬부기가 파이리를 그리워하고 있다... 🥲")

**왜:** Aibo의 감정 진화 + Character.AI의 "관계가 깊어지면 기억이 늘어난다" 원리. "내 꼬부기랑 파이리 짝꿍이야"라는 유저 자랑이 소셜 콘텐츠가 된다.

**구현:** 공동 배치 일수 `camp_placements.placed_at` 기반 계산. `camp_relationships` 테이블. 관계별 소식 템플릿 15개+.

### 8. 캠프 방문자 (레어 이벤트) 🅱️
⭐⭐⭐⭐ 실현 | 🔥🔥🔥🔥🔥 참여 | 💎💎💎 독창

**무엇:** 1~2일에 1회 빈도로 희귀 포켓몬이 캠프를 방문.

```
✨ 뮤가 캠프 근처에 나타났다!
🏕 3분 내 5명이 반응하면 머물러요!
   [환영하기 🤗] 현재 2/5명
```

성공 시: 전원 조각 +5 + 해당 포켓몬 도감 기록 (소유 아님)
실패 시: "뮤가 떠났다... 다음에 또 올 거예요."

**왜:** 포켓몬GO 레이드의 텍스트 축소판. Variable Ratio Reinforcement — 가장 강력한 참여 메커니즘. "우리가 불렀다!"의 공동 달성감.

**구현:** `camp_visitors` 테이블 + 스케줄러 (확률 기반 등장) + 반응 카운팅 + 타임아웃 처리.

### 9. 캠프 엽서 (주간 비주얼 리포트) 🅱️
⭐⭐⭐ 실현 | 🔥🔥🔥🔥 참여 | 💎💎💎💎 독창

**무엇:** 주 1회 (일요일 오후) 캠프의 한 주를 요약한 이미지 카드 생성.

이미지 구성:
- 배경: 현재 캠프 레벨에 맞는 배경 (빈 터 ~ 포켓몬 왕국)
- 배치된 포켓몬 스프라이트 6마리
- 주간 통계: 조각 수집량, 이벤트 참여 수, MVP
- 캠프 레벨 프로그레스바

```
📸 이번 주 캠프 엽서가 도착했어요!
🏛️ [크립토라운지] 캠프 Lv.10 — 광장
이번 주 조각: 47개 | 이벤트: 5회 | MVP: 문유

**왜:** 워들의 "결과 공유"를 시각적으로 확장. SNS/다른 방에 자랑 가능한 바이럴 콘텐츠. `camp_card_generator.py`가 이미 존재하지만 사용되지 않고 있음 — 이걸 활성화하는 자연스러운 경로.

**구현:** 기존 `camp_card_generator.py` 활용 + 주간 통계 쿼리 + 일요일 스케줄러.

### 10. 포켓몬 친밀도 → 미세 배틀 보너스 🅱️
⭐⭐⭐⭐⭐ 실현 | 🔥🔥🔥 참여 | 💎💎💎 독창

**무엇:** 캠프 7일+ 배치 포켓몬이 배틀에서 미세 보너스.

| 배치 기간 | 효과 |
|----------|------|
| 7일 | 데미지 +1% |
| 14일 | 데미지 +2%, 회피 +1% |
| 30일 | 데미지 +3%, 회피 +2%, 치명타 +1% |

**왜:** 포켓몬 공식 시리즈의 친밀도 배틀 보너스. 라이트(캠프) ↔ 배틀 유저 사이 브릿지. **결정적이지 않아 의무감을 주지 않되, 있으면 좋은 수준.**

**구현:** `battle_service._resolve_battle()`에서 `camp_placements.placed_at` 기반 일수 체크 → 보정값 적용. **코드 10줄.**

## 🎯 핵심 전략 요약

### 캠프의 3개 기둥

```
1. 감정 (Emotion)
   포켓몬 일기 → 파트너 시스템 → 짝꿍 → 편지
   "내 포켓몬이 살아있다"

2. 소셜 (Social)
   날씨 대화 → 번들 협력 → 예측 베팅 → 방문 교류
   "같이 하니까 재밌다"

3. 성장 (Growth)
   조각 수집 → 이로치 전환 → 타입 마스터리 → 역할 분화
   "매일 조금씩 나아진다"
```

### Phase 2에서 검증할 가설

1. **날씨가 정말 대화를 만드는가?** → 날씨 도입 후 채팅방 메시지 수 변화 측정
2. **일기가 DM 조회를 늘리는가?** → `내캠프` 호출 빈도 변화 측정
3. **복귀 메시지가 이탈을 줄이는가?** → 3일+ 미접속 후 복귀율 변화 측정

### 절대 잊지 말 것

> **"채팅방을 더럽히지 않는다"** — 하루 캠프 메시지 5개 이하
> **"벌주지 않는다"** — 안 해도 깎이지 않는다
> **"버튼 하나"** — 유저의 능동적 행동은 최소한으로
> **"같은 문제, 같은 대화"** — 모두가 같은 걸 보고 반응한다
