# 배틀 상성 & 레어리티 개편 기획서

## 개요
현재 배틀 상성 배율이 너무 약하고(1.3x/0.7x), 레어리티가 종족값과 무관하게 수동 배정됨.
실제 포켓몬 게임에 맞게 전면 개편.

---

## 1. 상성 배율 변경

### 변경 내용

| 상황 | 현재 | 변경 후 | 실제 포켓몬 |
|------|------|---------|-----------|
| 효과적 (물→불) | 1.3x | **2.0x** | 2.0x |
| 별로 (불→물) | 0.7x | **0.5x** | 0.5x |
| 면역 (노말→고스트) | 0.3x | **0x** | 0x |

### 수정 파일: `config.py`
```python
BATTLE_TYPE_ADVANTAGE_MULT = 2.0      # 1.3 → 2.0
BATTLE_TYPE_DISADVANTAGE_MULT = 0.5   # 0.7 → 0.5
```

### 수정 파일: `utils/battle_calc.py`
면역 시 0.3x → 0x (완전 무효) 변경

### 영향 분석
- 상성 유리 시 **데미지 2배** → 팀 빌딩의 중요성 대폭 상승
- 면역 타입은 **데미지 0** → 고스트/페어리 등 전략적 가치 급상승
- 속도/크리 운빨로 상성 뒤집기 어려워짐 → 정상적인 배틀 환경

### 상성 차트 (변경 없음, 배율만 조정)
```
불꽃 → 풀, 얼음, 벌레, 강철 (2.0x)
물   → 불꽃, 땅, 바위 (2.0x)
풀   → 물, 땅, 바위 (2.0x)
전기 → 물, 비행 (2.0x)
...
```

## 4. 게임 밸런스 영향

### 긍정적
- ✅ **상성 전략 중요해짐**: 2.0x/0.5x면 팀 구성이 핵심
- ✅ **진화 보상 명확**: 파이리(커먼) → 리자드(레어) → 리자몽(에픽) 자연스러운 진행
- ✅ **거래소 활성화**: 상성 커버를 위한 타입 다양화 수요 증가
- ✅ **공정한 분류**: 윈디(555) 같은 강포켓몬이 드디어 에픽

### 주의사항
- ⚠️ **피카츄 커먼 강등**: 인지도 높은 포켓몬이지만 종족값은 320 (라이츄가 485로 레어)
- ⚠️ **스타터 커먼 강등**: 파이리/꼬부기 등이 커먼으로 내려감 (하지만 최종진화는 에픽)
- ⚠️ **면역 0x**: 노말→고스트 등 데미지 0이 됨 → 전략적이지만 극단적일 수 있음
- ⚠️ **기존 유저 영향**: 이미 잡은 포켓몬의 레어리티가 바뀜 (도감/통계에 영향)

## 5. 구현 범위

### 수정 파일

| 파일 | 작업 |
|------|------|
| `config.py` | 상성 배율 2.0/0.5 변경 |
| `utils/battle_calc.py` | 면역 0x 처리 변경 |
| `models/pokemon_data.py` | 251종 레어리티 재배정 |
| `database/seed.py` | 레어리티 업데이트 쿼리 추가 (기존 DB 반영) |

### DB 마이그레이션
기존 `pokemon_master` 테이블의 rarity 값을 UPDATE 해야 함.
새로 시드하는 게 아니라 **UPDATE 쿼리**로 변경 (유저 데이터 보존).

```sql
-- 예시: 윈디 레어→에픽
UPDATE pokemon_master SET rarity = 'epic' WHERE id = 59;
-- 예시: 파이리 에픽→커먼
UPDATE pokemon_master SET rarity = 'common' WHERE id = 4;
```

### 봇 재시작 필요
- 포켓몬 캐시가 메모리에 있으므로 재시작 후 반영

## 6. 포획률/출현율 (변경 없음)

현재 설정이 합리적이므로 유지:
- 커먼: 출현 50%, 포획 70%
- 레어: 출현 30%, 포획 40%
- 에픽: 출현 15%, 포획 15%
- 레전더리: 출현 5%, 포획 3%

에픽이 47종으로 늘어나므로 개별 에픽 포켓몬의 출현 빈도는 약간 낮아지지만,
에픽 전체 출현율은 동일(15%)하므로 체감 차이 미미.
