  • 디스코드 뽀삐봇 (19만 서버): 가상재화 도박이 압도적 인기. 한국 디스코드 리스트

실패 패턴

  • 탭투언 피로: Hamster Kombat 86% 이탈. 게임성 없는 반복 = 3개월 수명
  • 이벤트 피로: 의무감 되면 끝
  • 개인 플레이: 소셜 상호작용 없으면 활성화 효과 제로
핵심: 채팅방에서 함께 보고 반응하는 것 자체가 콘텐츠.

이벤트 채널 등록 (옵트인 방식)

기능 업데이트 시

  • channel_owner 구독자 전원에게 DM: "매일 이벤트 기능 추가! 참여하시겠습니까?"
  • 참여 → 채널 초대링크 입력 안내
  • 미참여 → 스킵 (나중에 채널등록으로 언제든 가능)

등록

  • 채널등록 (DM) → 채널 초대링크 입력 → 완료
  • 자기 채널 아니어도 OK — 원하는 채널 링크 자유 지정
  • 1구독 = 1채널
  • 초대링크 변경: 채널수정 (DM)으로 언제든 가능

해제

  • 채널해제 (DM) 또는 구독 만료 시 자동

Phase 1: 퀴즈만 먼저 구현. 배틀로얄은 Phase 2.

미니게임 1: 포켓몬 퀴즈

포켓몬 퀴즈 (Pokemon Quiz)
"포켓몬 박사 대결"

퀴즈 유형 (매일 랜덤) — 1세대(No.1~151) 한정

유형출제예시
실루엣포켓몬 실루엣 이미지[검은 실루엣] → "리자몽!"
타입타입 조합 질문"불꽃+비행 포켓몬은?" → "리자몽!"
도감 설명도감 텍스트"꼬리 불꽃이 꺼지면..." → "파이리!"

플레이

  • 5문제 연속 출제, 문제당 30초 (총 약 3분)
  • 매 문제: 실루엣 이미지 → 정답 입력 → 선착순 5명만 정답 인정 → 다음 문제
  • 5문제 종료 후 일괄 정산
  • 틀린 답 무시, 6번째 이후 정답도 무시
  • 정답 공개: 실루엣 → 컬러 이미지 전환
  • 5문제 후 정답 수 합산으로 등수 (동점 시 응답 시간)
  • 0문제 맞춰도 참가 보상(200 BP) 지급

보상 (문제당, 5문제 후 일괄 정산)

항목보상
1등 정답 (최초 정답자)IV 선택리롤 2개
2~5등 정답IV 선택리롤 1개
6등~ 정답500 BP
참가 (0문제)500 BP
최대 (5문제 전부 1등)IV 선택리롤 10개

미니게임 2: 배틀로얄

배틀로얄 (Battle Royale)
"운빨 서바이벌"

플레이

  • 이벤트 시작 → 참가 버튼 (30초 모집, 최대 30명)
  • 봇이 라운드별 랜덤 전투 자동 진행
  • 매 라운드 2명 매칭 → 승자 생존, 패자 탈락
  • 참가자의 대표 포켓몬 이름 사용
  • 순수 랜덤 — 스탯/IV 무관. 잉어킹이 뮤츠 이김
  • 내레이션: 기술 사용 연출
  • 마지막 생존자 우승

별도: 폭탄돌리기 (상시 미니게임)

매일 이벤트와 별개. 채팅방에서 아무 때나 즐기는 상시 명령어.

폭탄 돌리기 (Hot Potato)
"나한테 오지 마"

플레이

  • 폭탄 명령어 → 참가 버튼 (최소 3명, 최대 15명)
  • 폭탄이 랜덤 참가자에게 → 제한 시간 안에 패스
  • 시간 내 패스 못 하면 탈락
  • 매 라운드 시간 줄어듦: 10초 → 8초 → 6초 → 5초
  • 마지막 생존자 우승

보상

  • 우승 300BP, 2등 100BP (상시 미니게임이므로 가볍게)
  • 일일 참가 5회 제한

구현 범위

DB

  • event_channels — 이벤트 등록 채널
  • daily_events — 일일 이벤트 기록 (유형, 채널, 상태)
  • event_participants — 참가자 기록 (등수, 점수, 보상)

파일

  • handlers/group_event.py — 이벤트 핸들러
  • services/event_service.py — 퀴즈 + 배틀로얄 엔진
  • jobs/event_jobs.py — 매일 스폰 스케줄러
  • database/event_queries.py — DB 쿼리
  • handlers/group_bomb.py — 폭탄돌리기 (Phase 2)

난이도

M — 2종 게임 엔진 + 스폰 시스템 + DM 알림 + 보상
