01 컨셉 요약

1마리 포켓몬으로 도전하는 로그라이크 던전

매 층 랜덤 적과 자동 배틀 → 승리 시 버프 3개 중 택1 (뱀서 스타일).
버프가 쌓일수록 포켓몬이 괴물로 성장. 어디까지 갈 수 있나?
투자한 포켓몬(이로치 + S급 IV + 고친밀도)일수록 고층 도달.
매일 던전 테마가 바뀜 → 다양한 포켓몬 사용 유도. 입장권 필요 → 긴장감.

핵심 키워드

🎮 1마리 — 팀 구성 고민 없이 즉시 참여
🎲 랜덤 적 — 매 런 다른 경험, 상성 운 요소
⬆️ 버프 선택 — 뱀서 레벨업 느낌, 플레이어 전략
🎫 입장권 — 자원 관리, 매일 접속 동기
💪 육성=층수 — 이로치/IV/친밀도 투자의 목적지
🏆 랭킹 — 엔드게임 경쟁, 소셜 화제

해결하는 문제

현재 문제던전이 해결
❌ DAU 200 이하 하락✅ 일일 무료 입장권 → 매일 접속 동기
❌ 캠프 참여율 0%✅ 이로치=층수, 조각 수요 폭증
❌ 엔드게임 할 것 없음✅ 랭킹 경쟁, 마일스톤 칭호
❌ 커먼/레어 쓸모없음✅ 코스트별 버프 차등 → 커먼 역전 가능
❌ IV/이로치 투자 동기 없음✅ S급 이로치 = 10층+ 차이

이벤트 층 — 상점/도박/보물 (10% 확률)

03 던전 테마 (일일 랜덤)

매일 자정(KST) 테마가 바뀜. 테마에 따라 적 타입 분포와 보너스 버프가 달라짐.

테마적 타입 비중유리한 포켓몬보너스 버프
🌿 고대 숲풀/독/벌레 60%불/비행/얼음불꽃 계열 ↑
🔥 화산불/드래곤/격투 60%물/땅/바위물 계열 ↑
🌊 심해물/얼음/비행 60%전기/풀전기 계열 ↑
⚡ 발전소전기/강철/노말 60%땅/격투/불땅 계열 ↑
🪨 동굴땅/바위/고스트 60%물/풀/얼음얼음 계열 ↑
🔮 신전에스퍼/악/페어리 60%악/강철/독악 계열 ↑
🐉 용의 둥지드래곤 80%페어리/얼음주말 특수 (토~일)
💡 "오늘 화산이네, 갸라도스로 가야지" → 매일 다른 전략
⚠️ 5층 보스는 테마 벗어날 수 있음 → 방심 불가

04 적 생성 & 난이도 스케일링

4-1. 적 포켓몬 생성

• 테마 타입 풀에서 60%, 기타 타입에서 40% 확률 랜덤 선택
• 적 희귀도: 층수에 비례 (1~5층 커먼 → 6~15층 레어 혼합 → 16+ 에픽 혼합)
• 적 IV: 0 고정 (순수 스케일링으로 난이도 조절)

09 포켓몬 스탯 반영

기본 배틀 스탯 그대로 사용 (기존 calc_battle_stats) = 종족값 × 성격배율 × (1 + 친밀도×0.04) × 진화단계 × IV보정 × 희귀도보정

투자에 따른 차이

상태실효 배율예상 한계층
커먼 C급 일반×1.0 (기준)~10층
레어 A급 일반×1.15~15층
에픽 A급 이로치×1.55~22층 (버프 2층마다)
전설 S급 이로치×1.75~28층 (버프 3층마다)

11 보상 루프 요약

🔄 일일 루프

던전 도전 → BP + 테마 조각 획득
→ 조각으로 이로치 전환 (캠프)
→ 이로치 포켓몬으로 더 높은 층
→ 더 많은 보상

🔄 주간 루프

주간 랭킹 → 결정/입장권/마볼 보상
→ 결정으로 에픽+ 이로치 전환
→ 전투력 상승
→ 다음 주 더 높은 순위

🔄 시즌 루프

시즌 마일스톤 → IV 스톤/만능 조각
→ IV 최적화
→ 영구 전투력 상승
→ 다음 시즌 더 높은 기록

🔄 육성 루프

포획 (좋은 IV) → 밥/놀기 (친밀도)
→ 캠프 (조각) → 이로치 전환
→ 던전 도전 → 보상
→ 더 나은 포켓몬 육성
