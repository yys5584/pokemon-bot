# 🔨 이로치 분해 시스템 기획서

> 작성일: 2026-03-13
> 콘셉트: 팰월드식 "귀여운데 잔인한" 자원 순환
> 전제: 캠프 Phase 1 코어 위에 얹는 확장

---

## 1. 왜 필요한가

### 현재 문제

| 문제 | 설명 |
|------|------|
| 조각의 용도 부족 | 이로치 전환만 있으면 "이로치 화폐"로 끝 |
| 이로치 과잉 | 파워유저는 이로치 200마리+, 쓸 데 없음 |
| 캠프 성장 한계 | 레벨 20 찍으면 할 게 없어짐 |
| 배틀↔캠프 단절 | 두 시스템이 완전히 분리되어 있음 |

### 해결

**이로치를 갈아서 "결정"을 얻고, 결정으로 캠프 시설을 짓는다.**

```
이로치 포켓몬 → [분해] → 💎 타입 결정
                          ↓
            캠프 시설 건설/업그레이드
            특수 구역 해금
            캠프 장식/테마
            전체 버프 효과
```

## 2. 결정 시스템

### 2-1. 결정 종류 (= 포켓몬 타입)

분해 시 해당 포켓몬의 **주 타입**에 맞는 결정 획득.

| 결정 | 타입 | 이모지 |
|------|------|--------|
| 초록 결정 | 풀/벌레 | 💎🟢 |
| 파랑 결정 | 물 | 💎🔵 |
| 빨강 결정 | 불 | 💎🔴 |
| 갈색 결정 | 바위/땅 | 💎🟤 |
| 하양 결정 | 비행/노말 | 💎⚪ |
| 보라 결정 | 악/고스트/독 | 💎🟣 |
| 노랑 결정 | 전기 | 💎🟡 |
| 분홍 결정 | 페어리/에스퍼 | 💎🩷 |
| 은색 결정 | 강철/얼음 | 💎🩶 |
| 금색 결정 | 드래곤/격투 | 💎🥇 |

복합 타입은 주 타입 결정 + 부 타입 결정 소량 추가.

### 2-2. 등급별 결정 수량

| 이로치 등급 | 주 타입 결정 | 부 타입 결정 | 특수 재료 |
|------------|------------|------------|----------|
| ⭐ 커먼 | 3 | 1 | - |
| ⭐ 레어 | 5 | 2 | - |
| ⭐ 에픽 | 8 | 3 | ✨ 무지개조각 ×1 |
| ⭐ 전설 | 12 | 5 | ✨ 무지개조각 ×3 |
| ⭐ 초전설 | 15 | 7 | ✨ 무지개조각 ×5 + 🌟 전설의파편 ×1 |

**무지개조각**: 아무 결정 대체 가능한 와일드카드
**전설의파편**: 최상위 시설에만 쓰이는 초레어 재료

### 2-3. 보너스 조건

| 조건 | 보너스 |
|------|--------|
| 캠프 배치 7일+ 이로치 | 결정 +2 (정이 들었는데...) |
| 캠프 배치 30일+ 이로치 | 결정 +5 + "유언" 특별 대사 |
| 친밀도 MAX 이로치 | 결정 +3 + 기념 칭호 해금 |
| S랭크 이로치 (합성 최고등급) | 결정 ×2배 |

## 3. 캠프 시설 시스템

### 3-1. 시설 종류

결정으로 캠프에 시설을 건설. 시설은 **영구 효과** 제공.

#### 🏗️ 기본 시설 (캠프 Lv.5+)

| 시설 | 비용 | 효과 | 업그레이드 |
|------|------|------|-----------|
| 🌿 온실 | 초록 ×10 | 풀숲 조각 +1/일 | Lv.2: +2, Lv.3: +3 |
| 💧 양어장 | 파랑 ×10 | 호수 조각 +1/일 | Lv.2: +2, Lv.3: +3 |
| 🔥 대장간 | 빨강 ×10 | 모닥불 조각 +1/일 | Lv.2: +2, Lv.3: +3 |
| 🏥 포켓몬센터 | 아무 결정 ×15 | 배치 슬롯 +1 | Lv.2: +2 (최대) |
| 🍳 식당 | 아무 결정 ×8 | 이벤트 보상 +1조각 | Lv.2: +2 |

#### ⚔️ 전투 시설 (캠프 Lv.10+)

| 시설 | 비용 | 효과 |
|------|------|------|
| 🏋️ 훈련장 | 금색 ×12 + 빨강 ×5 | 배치 포켓몬 배틀 데미지 +2% |
| 🛡️ 방어탑 | 은색 ×12 + 갈색 ×5 | 배치 포켓몬 배틀 방어 +2% |
| 📚 전략실 | 보라 ×10 + 분홍 ×5 | 배틀 시 스킬 2개 중 선택 가능 |

#### ✨ 특수 시설 (캠프 Lv.15+)

| 시설 | 비용 | 효과 |
|------|------|------|
| 🌈 무지개탑 | 무지개조각 ×10 + 각 색 결정 ×3 | 이로치 전환 비용 -20% |
| 🔮 예언의 동굴 | 보라 ×15 + 전설의파편 ×1 | 내일 임무 미리보기 |
| 🏯 드래곤 둥지 | 금색 ×20 + 전설의파편 ×2 | 특수 구역 "드래곤 둥지" 해금 |
| 🌊 심해 탐사선 | 파랑 ×20 + 전설의파편 ×2 | 특수 구역 "심해" 해금 |

#### 🎨 장식 시설 (아무 레벨)

| 시설 | 비용 | 효과 |
|------|------|------|
| 🎪 축제 텐트 | 아무 결정 ×5 | 캠프 소식에 축제 테마 추가 |
| 🌸 벚꽃나무 | 분홍 ×5 | 캠프 카드에 벚꽃 이펙트 |
| ⛲ 분수대 | 파랑 ×8 | 캠프 카드에 분수 이펙트 |
| 🗿 석상 | 갈색 ×8 | 분해한 포켓몬 기념비 (이름 표시) |

### 3-2. 시설 업그레이드

대부분 시설은 3단계까지 업그레이드 가능.
업그레이드마다 비용 1.5배, 효과 누적.

```
온실 Lv.1: 초록 ×10 → 풀숲 +1
온실 Lv.2: 초록 ×15 → 풀숲 +2 (누적 +3)
온실 Lv.3: 초록 ×20 → 풀숲 +3 (누적 +6)
```

### 3-3. 시설 현황 표시

DM `내캠프`에서:

```
🏕 캠프 시설 현황
━━━━━━━━━━━━━
🌿 온실 Lv.2 — 풀숲 조각 +3/일
💧 양어장 Lv.1 — 호수 조각 +1/일
🏥 포켓몬센터 Lv.1 — 슬롯 +1
🍳 식당 Lv.1 — 이벤트 보상 +1

💎 보유 결정: 🟢×4 🔵×2 🔴×7 🟤×0
✨ 무지개조각: 2개
━━━━━━━━━━━━━
   [시설 건설🔨]
```

## 4. 분해 UX — 팰월드 감성

### 4-1. 분해 플로우

DM에서 `분해` 또는 `이로치분해` 입력.

```
🔨 이로치 분해소
━━━━━━━━━━━━━
이로치를 분해하여 캠프 건설 재료를 얻습니다.

⚠️ 분해한 포켓몬은 영구히 사라집니다.

분해 가능한 이로치: 42마리
   [분해할 포켓몬 선택]
```

### 4-2. 포켓몬 선택 후

```
🔨 분해 대상: ✨ 꼬부기 (물)
━━━━━━━━━━━━━

예상 획득:
💎🔵 파랑 결정 ×5
💎⚪ 하양 결정 ×2

정말 분해하시겠습니까?
   [분해하기 🔨]  [취소]
```

### 4-3. 분해 실행 — 여기가 핵심

**단계별 연출:**

```
🔨 분해 중...

꼬부기: "주인... 나 어디가는 거야...?"
```

(1초 후 메시지 수정)

```
🔨 분해 완료

꼬부기: "...주인이 원하는 거라면... 괜찮아... 😢"

💎🔵 파랑 결정 ×5 획득!
💎⚪ 하양 결정 ×2 획득!

꼬부기의 등껍질이 아름다운 결정이 되었습니다.
🗿 캠프에 작은 기념비가 세워졌습니다.
```

## 5. 특수 구역

결정으로 해금하는 상위 구역. 기존 7구역 위에 추가.

| 구역 | 해금 비용 | 조각 색 | 특징 |
|------|----------|--------|------|
| 🐉 드래곤 둥지 | 금색 ×20 + 전설의파편 ×2 | 금색 | 드래곤 타입 전용, 전설 조각 확률 UP |
| 🌊 심해 | 파랑 ×20 + 전설의파편 ×2 | 심해색 | 수중 포켓몬 전용, 레어 이벤트 |
| ❄️ 얼음동굴 | 은색 ×15 + 무지개 ×5 | 은색 | 얼음 타입, 겨울 한정→상시 |
| 🌙 달빛숲 | 보라 ×15 + 분홍 ×10 | 달빛색 | 페어리/에스퍼, 야간 보너스 2배 |

특수 구역은 **그룹 전체 공유**. 누군가 결정을 투자해서 지으면 방 전체가 사용.

```
🐉 드래곤 둥지 해금!

🏕 [크립토라운지] 캠프에 드래곤 둥지가 열렸습니다!
문유님이 금색 결정 20개 + 전설의파편 2개를 투자했습니다.

이제 드래곤 타입 포켓몬을 배치할 수 있습니다!
━━━━━━━━━━━━━
🐉 드래곤 둥지: 망나뇽 (공격 20+)   ← 새 임무!
```

→ "문유가 드래곤 둥지 열어줬다!" = 기여자에 대한 존경 + 소셜

## 6. 경제 밸런스

### 6-1. 이로치 → 결정 → 시설 흐름

```
커먼 이로치 1마리 = 결정 ~4개
레어 이로치 1마리 = 결정 ~7개
에픽 이로치 1마리 = 결정 ~11개 + 무지개 1

기본 시설 1개 = 결정 ~10개 (커먼 이로치 2~3마리)
전투 시설 1개 = 결정 ~17개 (레어 이로치 2~3마리)
특수 구역 1개 = 결정 ~20개 + 전설의파편 (초전설 이로치 1마리+)
```

### 6-2. 시설 전체 건설 비용 추산

| 시설 그룹 | 총 결정 필요 | 이로치 환산 |
|----------|------------|-----------|
| 기본 시설 5개 (Lv.1) | ~55 | 커먼/레어 ~10마리 |
| 기본 시설 전부 Lv.3 | ~170 | ~30마리 |
| 전투 시설 3개 | ~55 | ~10마리 |
| 특수 구역 4개 | ~100 + 파편 8 | 에픽/전설 ~10마리 |
| 장식 전부 | ~30 | ~6마리 |
| **전체 맥스** | **~410 + 파편 8** | **~60~70마리** |

→ 파워유저(이로치 200마리)도 한 번에 다 못 지음. 선택과 집중 필요.
→ 일반유저는 조각→이로치→분해 사이클로 천천히 성장.
→ 그룹 단위 투자이므로 여러 명이 분담 가능.

### 6-3. 조각 vs 결정 구분

| | 조각 (Fragment) | 결정 (Crystal) |
|--|----------------|----------------|
| 획득 | 캠프 소식/이벤트 (매일 자동) | 이로치 분해 (능동) |
| 용도 | 이로치 전환 | 시설 건설 |
| 교환 | 유저간 교환 가능 (추후) | 교환 불가 (투자 전용) |
| 느낌 | "매일 조금씩 모으는 일상" | "큰 결심으로 얻는 귀한 재료" |

→ **두 화폐의 역할이 명확히 분리됨.** 조각은 개인 보상, 결정은 캠프 투자.

## 7. 기존 시스템 연동

### 7-1. 합성(갈갈이)과의 관계

현재 합성: 같은 포켓몬 2마리 → 상위 등급 1마리.
이로치끼리 합성하면 S랭크 이로치.

**분해와의 시너지:**
- 이로치 2마리 중 1마리는 합성용, 1마리는 분해용 → 전략적 선택
- S랭크 이로치를 분해하면 결정 ×2배 → 고급 시설용
- "갈갈이할까 분해할까" 고민 자체가 컨텐츠

### 7-2. 거래소 연동

- 거래소에서 이로치 구매 → 분해용으로 사는 수요 발생
- "커먼 이로치 3000BP에 삽니다" 같은 새로운 거래 패턴
- 결정 자체는 거래 불가 (캠프 투자 목적이므로)

### 7-3. 배틀 연동 (전투 시설)

- 훈련장/방어탑: 캠프에 배치된 포켓몬이 배틀에서 소폭 보너스
- 캠프 투자 → 배틀 성능 향상이라는 간접 경로
- "캠프 안 하던 배틀러도 시설 보너스 때문에 관심"

## 9. 구현 범위

### DB 테이블 추가

```sql
-- 결정 보유량
CREATE TABLE camp_crystals (
  user_id BIGINT NOT NULL,
  crystal_type VARCHAR(20) NOT NULL,  -- 'green', 'blue', 'red', ...
  amount INT DEFAULT 0,
  PRIMARY KEY (user_id, crystal_type)
);

-- 분해 로그
CREATE TABLE camp_decompose_log (
  id SERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL,
  pokemon_name VARCHAR(50) NOT NULL,
  pokemon_type VARCHAR(20) NOT NULL,
  rarity VARCHAR(20) NOT NULL,
  was_shiny BOOLEAN DEFAULT TRUE,
  camp_days INT DEFAULT 0,           -- 캠프 배치 일수
  crystals_gained JSONB NOT NULL,    -- {"blue": 5, "white": 2}
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 캠프 시설
CREATE TABLE camp_facilities (
  chat_id BIGINT NOT NULL,
  facility_type VARCHAR(30) NOT NULL,
  level INT DEFAULT 1,
  built_by BIGINT NOT NULL,          -- 건설한 유저
  built_at TIMESTAMPTZ DEFAULT NOW(),
  PRIMARY KEY (chat_id, facility_type)
);

-- 기념비 (분해된 포켓몬 기록)
CREATE TABLE camp_memorial (
  id SERIAL PRIMARY KEY,
  chat_id BIGINT NOT NULL,
  user_id BIGINT NOT NULL,
  pokemon_name VARCHAR(50) NOT NULL,
  camp_days INT DEFAULT 0,
  decomposed_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 수정 파일

```
config.py                    — CAMP_CRYSTALS, CAMP_FACILITIES 상수 추가
database/camp_queries.py     — 결정/시설/분해 쿼리 추가
services/camp_service.py     — 분해 로직, 시설 건설/효과 로직
handlers/dm_camp.py          — 분해 DM 핸들러, 시설 조회
handlers/camp.py             — 시설 건설 (그룹), 기념비 조회
```

### 구현 순서

```
[Step 1] 분해 기본 — 2~3일
  ☐ 결정 테이블 + 분해 로그
  ☐ DM 분해 핸들러 (선택 → 확인 → 연출 → 결정 지급)
  ☐ 등급별 결정 수량 로직
  ☐ 분해 대사 템플릿 (성격 × 배치기간)

[Step 2] 시설 시스템 — 3~4일
  ☐ 시설 테이블 + 쿼리
  ☐ 그룹 시설 건설 핸들러 (결정 소모)
  ☐ 시설 효과 적용 (조각 보너스, 슬롯, 이벤트 보상)
  ☐ DM 시설 현황 조회

[Step 3] 감성 연출 — 1~2일
  ☐ 캠프 소식에 분해 반응 ("빈자리에 꽃을 놓았다")
  ☐ 기념비 시설 + 조회
  ☐ 다음 소식 피카츄 불안 반응 등

[Step 4] 특수 구역 — 2~3일
  ☐ 드래곤 둥지 / 심해 / 얼음동굴 / 달빛숲
  ☐ 구역별 임무 + 조각 + 이벤트 템플릿
  ☐ 해금 공지 연출
```

## 10. 핵심 원칙

> **"귀여운데 잔인한"** — 분해 과정에서 감정을 느끼게 하되, 결과물은 충분히 가치있게.
>
> **"선택과 집중"** — 이로치 200마리 있어도 뭘 갈지 고민하게. 전부 못 지으니까.
>
> **"기여 = 존경"** — 초전설 갈아서 드래곤 둥지 연 사람은 영웅. 소셜 자본.
>
> **"죄책감 = 콘텐츠"** — 스크린샷 찍어서 방에 올리고 "ㅋㅋㅋ 미안해 꼬부기야" 하는 순간이 진짜 engagement.
