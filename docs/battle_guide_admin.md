1. **노라키(노말)**: 고스트 타입에 면역. 상대에 고스트가 있으면 무력화
2. **깜까미(BST 380)**: 코스트 1이라 싸지만 체급 차이가 큼. 코뿌리 지진에 3턴 컷
3. **루기아 선봉**: 초전설을 선봉에 넣으면 상대 카운터에 바로 잡힐 수 있음
4. **썬더+루기아 비행 겹침**: 얼음/바위에 동시에 약함

## 결론

**추천 팀 (안 A)**: 누오 → 마기라스 → 릴리요 → 루기아 → 노라키 → 깜까미

이유:
1. 누오 선봉으로 전기 면역 + 물타입 안정
2. 마기라스(IV 165)가 보유 중 **가성비 최강** (BST 600 + IV 165 = 765, 코스트 4)
3. 릴리요가 물/땅 타입 카운터
4. 루기아를 4번에 놓아서 상대가 카운터 소진한 후 투입
5. 노라키/깜까미는 체력 소모용 버퍼
