2. 설계 원칙

2-1. 코스트당 밸류 균등화

"코스트 1을 더 쓰면, 그만큼의 가치를 얻어야 한다"

목표: 1v1에서 높은 등급이 이기지만, 코스트 효율은 비슷하게.

핵심 공식:

코스트 N의 포켓몬 ≈ 코스트 1 포켓몬 N마리의 합산 가치
→ 전설(코스트5) ≈ 일반 5마리가 순차로 싸운 것과 비슷한 총 딜
→ 하지만 1v1에서는 전설이 일반을 원턴킬

2-2. 등급 = 확실한 벽

"한카리아스(에픽)와 스이쿤(전설)이 싸우면 스이쿤이 이기거나 비긴다 (상성이 나쁘지 않다면)"

  • 1v1: 상위 등급이 항상 승리 (상성 무관하면)
  • 같은 등급: 상성 + IV + 타입이 결정
  • 등급 2단계 차이: 상성 불리해도 승리

2-3. 팀 빌딩의 선택지

"코스트를 생각하면 한카리아스가 더 좋은 거지"

팀 코스트 18, 6마리 제한:

팀 구성 코스트 총 전투력 장점
에픽×4 + 일반×2 4×4+1×2=18 6마리, 두꺼운 라인업
초전설+에픽×2+레어+일반×2 6+8+2+2=18 에이스+서포트
전설×2+에픽+레어×2+일반 10+4+4+1=19 ← 초과 불가능
전설+에픽×2+레어+일반×2 5+8+2+2+1=18 ← 5마리? 코스트 남음

전설(코스트5)의 딜레마: - 전설 1마리 = 에픽(4) + 일반(1) 두 자리를 먹음 - 전설이 에픽+일반 합산보다 확실히 강해야 쓸 이유가 생김 - 현재: 전설 딜 511 < 에픽 딜 468 + 일반 딜 126 = 594 → 안 씀

3. BST 리폼안

3-1. 등급별 목표 BST 상한

현재 BST를 건드리지 않되, 상한선(cap)을 적용. 상한 초과 포켓몬만 깎이고, 약한 포켓몬은 그대로.

등급 현재 BST 상한 영향받는 포켓몬
일반 180~450 400 체리꼬(450→400), 골덕(480→400 ← 레어로 승격?)
레어 285~535 480 덩쿠림보(535→480), 하마돈(525→480)
에픽 440~670 560 게을킹(670→560), 한카리아스(600→560), 망나뇽(600→560)
전설 480~730 660 레지기가스(730→660)
초전설 600~720 없음 변경 없음

적용 방식: 비례 스케일링

if pokemon_bst > cap:
    scale = cap / pokemon_bst  # 예: 600/670 = 0.836
    각 스탯 = int(원래 스탯 * scale)

원래 스탯 비율은 유지 (공격형은 여전히 공격형).

5. 팀 빌딩 시나리오

시나리오 A: 에픽 풀팀 (현 메타)

에픽×4 + 레어×1 = 코스트 18
한카리아스(560) + 메타그로스(560) + 망나뇽(560) + 마기라스(560) + 하마돈(480)
총 전투력: ~7,200

시나리오 B: 전설 혼합 (리폼 후 가능)

전설×1 + 에픽×2 + 레어×1 + 일반×2 = 코스트 5+8+2+2=17 (1 여유)
스이쿤(580) + 한카리아스(560) + 메타그로스(560) + 하마돈(480) + 피카츄(315) + 토게틱(340)
총 전투력: ~5,800

시나리오 C: 초전설 에이스

초전설×1 + 에픽×1 + 레어×2 + 일반×2 = 코스트 6+4+4+2+2=18
뮤츠(680) + 한카리아스(560) + 하마돈(480) + 눈설왕(480) + 피카츄(315) + 토게틱(340)
총 전투력: ~6,400

분석

팀 총 전투력 에이스 파워 라인업 두께
A: 에픽 풀 7,200 560 (보통) 두꺼움 (5마리 다 강함)
B: 전설 혼합 5,800 580 (높음) 약함 (하위 2마리 약함)
C: 초전설 에이스 6,400 680 (최강) 중간

A(에픽풀)가 총 전투력 최고 — 코스트 효율이 좋으니까. C(초전설)는 뮤츠가 2~3마리 킬 — 에이스 전략. B(전설)는 스이쿤이 에픽팀 1마리를 확실히 잡고 나머지 라인업으로 버팀.

→ 세 팀 다 이길 수 있는 구조 = 건강한 메타

7. FAQ

Q: 내 게을킹이 너프되는데? A: 에픽 등급에 맞는 전투력으로 조정. 대신 전설/초전설이 확실히 강해져서 등급의 의미가 생김.

Q: 전설을 써야 하는 이유? A: 1v1에서 에픽을 확실히 이김. 에이스로 쓰면 2~3마리 킬 가능. 팀에 전설 1마리 + 일반 필러 vs 에픽 풀팀은 대등한 승부.

Q: 일반 포켓몬은 쓸모없어지나? A: 코스트 1이라 코스트 효율은 최고. 팀 채우기에 필수. 상성 맞으면 충분히 활약.

Q: 초전설은 너무 강하지 않나? A: 코스트 6(팀의 1/3). 초전설 넣으면 나머지 5마리가 약해짐. 초전설 1마리가 3킬 해도 나머지가 무너지면 짐.

Q: 게을킹/레지기가스는? A: BST 상한 적용 안 함. 본가에서 게으름 특성 때문에 높은 BST를 가진 거라, 상한을 씌우면 이중 너프. 대신 2턴마다 빈둥빈둥하는 페널티가 밸런스를 잡아줌.
