# 주간보스 (Weekly Boss Raid) 기획서

## 개요
매주 등장하는 강력한 보스 포켓몬을 전체 유저가 협력해서 공격하는 비동기 레이드 시스템.

## 핵심 규칙

| 항목 | 내용 |
|------|------|
| 주기 | 월요일 00:00 ~ 일요일 23:59 (KST) |
| 공격 횟수 | 하루 1회 |
| 참여 위치 | 봇 DM (`보스` 명령) |
| 전투 방식 | 내 배틀팀 6마리 vs 보스 1마리 (턴제, 자동전투) |
| 보스 HP | 고정 (전체 유저 공유, 다같이 깎기) |
| 보스 처치 | 잡으면 추가 보너스. 못 잡아도 일요일 종료 |

## 보스

### 등장 방식
- 매주 월요일 자동 생성
- 전설/초전설 포켓몬 중 랜덤 (직전 주와 중복 방지)
- 속성은 해당 포켓몬의 원래 속성

### 보스 스탯
- **HP**: 500,000 (조정 가능, DAU 20~40명 기준)
- **공격/방어**: 초전설급 × 높은 배율
- **속성 약점**: 원작 상성 그대로 적용

### HP 세팅 근거
- 초전설 풀강 팀 → 1회 공격 딜량 ~3,000~5,000
- 일반 12코 팀 → 1회 공격 딜량 ~1,500~3,000
- DAU 20명 × 7일 = 140타 → 예상 총딜 350,000~700,000
- **500,000이면 간신히 잡히는 수준**

## 전투

### 방식
- 던전 자동전투 엔진 재활용
- 내 배틀팀 6마리가 순서대로 보스와 싸움
- 전멸하거나 턴 제한(60턴)까지의 **총 딜량**을 기록
- 상성 유리한 팀 구성이 핵심 전략

### 보스 행동
- 매턴 공격 (일반공격 / 속성공격 / 풀어택 랜덤)
- 방어/회복 없음 (샌드백 컨셉)

## 보상

### 1. 데일리 마일스톤 (매일 즉시 지급)
공격 직후 딜량에 따라 보상. **누적 아님, 그날 딜량 기준.**

| 딜량 | 보상 |
|------|------|
| 500+ | 30 BP |
| 1,000+ | 50 BP |
| 2,000+ | 80 BP + 조각 2개 |
| 3,000+ | 100 BP + 조각 3개 |
| 5,000+ | 150 BP + 조각 5개 + IV리롤 1개 |

※ 최고 도달 구간만 지급 (중복X)

### 2. 주간 랭킹 (월요일 정산)
주간 누적 딜량 기준 순위별 보상.

| 순위 | 보상 |
|------|------|
| 🥇 1위 | 500 BP + IV+3 + 마스터볼 2개 |
| 🥈 2~3위 | 300 BP + IV+3 + 마스터볼 1개 |
| 4~10위 | 200 BP + 시간단축권 2개 |
| 11~30위 | 100 BP + 시간단축권 1개 |

### 3. 보스 처치 보너스
보스 HP를 0으로 만들면 **그 주 참여자 전원**에게 추가 보상:
- 마스터볼 1개 + 100 BP

## 주간 사이클

| 시점 | 이벤트 |
|------|--------|
| 월요일 00:10 | 지난주 랭킹 보상 정산 + 새 보스 생성 |
| 월~일 매일 | 하루 1회 공격 가능 |
| 보스 HP 0 | 처치 알림 + 참여자 보너스 (그 주 내 언제든) |
| 일요일 23:59 | 주간 종료 (못 잡았어도 끝) |

## 향후 확장 가능
- 보스별 특수 능력 (독 데미지, 반사 등)
- 보스 처치 시 한정 포켓몬 포획 기회
- 시즌 누적 랭킹
- 보스 난이도 자동 조절 (DAU 연동)
