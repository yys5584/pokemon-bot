# 포켓몬 던전 (로그라이크) 설계서

> 버전: v0.1 (2026-03-18)
> 상태: 설계 검토 중

## 컨셉

뱀서 스타일 로그라이크 던전. 포켓몬 1마리로 입장, 매 층 랜덤 적과 자동 배틀 후 버프 3개 중 택1.
투자한 포켓몬(이로치, 고IV, 고친밀도)일수록 고층 도달 가능. 랭킹 경쟁.

### 핵심 키워드
- **1마리** — 팀 구성 고민 없이 즉시 참여
- **랜덤 적** — 매 런 다른 경험, 상성 운 요소
- **버프 선택** — 뱀서 레벨업 느낌, 플레이어 전략
- **입장권** — 자원 관리, 긴장감
- **육성 = 층수** — 이로치/IV/친밀도 투자의 목적지

## 스테이지 설계

### 적 포켓몬 생성 규칙
- **매 런 랜덤** — 층마다 적 포켓몬 랜덤 배정
- 층수에 비례해 적 스탯 스케일링
- **5층마다 보스** (에픽+ 포켓몬, 스탯 1.5배)
- **10층마다 관장급** (특정 타입 통일, 스탯 2배) ← 포켓로그 관장 느낌

### 타입 상성
- 내 포켓몬 vs 적 포켓몬 타입 상성 그대로 적용
- 운 나쁘면 상성 불리한 적이 연속으로 → "이번 판은 재수없었다" → 다음 판 동기
- 운 좋으면 상성 유리 → 고층 돌파 쾌감

### 난이도 스케일링
```
적 스탯 = 기본스탯 × (1 + 층수 × 0.08)

예시 (커먼 기준):
  1층: ×1.08 (쉬움)
  10층: ×1.80 (보통)
  20층: ×2.60 (어려움)
  30층: ×3.40 (하드코어)
  50층: ×5.00 (극한)
```

## 버프 시스템

### 버프 등급
- **일반** (회색): 소폭 상승 (+10~15%)
- **고급** (파랑): 중폭 상승 (+20~30%)
- **희귀** (보라): 대폭 상승 or 특수 효과
- **전설** (금): 게임 체인저급 (10층+ 부터 등장)

### 버프 풀 (예시)

**공격계**
- 🗡 공격력 +15/25/40%
- 💀 치명타 확률 +10/20/30%
- 🔥 타입 일치 데미지 +20/35/50%
- ⚔️ 2연타 (20/35% 확률)

**방어계**
- ❤️ HP +15/25/40%
- 🛡 방어력 +15/25/40%
- 💚 매 턴 HP 3/5/8% 회복
- 🌀 피해 반사 10/20%

**유틸리티**
- 💨 선제공격 확률 +15/25/40%
- 🎯 명중률 +10/20% (회피 무시)
- 🍀 버프 등급 상향 (다음부터 고급 확률↑)
- 🔄 상성 불리 경감 50%

**전설급 (10층+)**
- 👻 1회 부활 (HP 30%로)
- 🌈 모든 타입 상성 유리
- 💎 스탯 전체 +25%
- ⏳ 적 선공 봉쇄 3층간

### 코스트별 버프 차등

| 희귀도 | 코스트 | 버프 주기 | 선택지 |
|--------|--------|----------|--------|
| 커먼 | 1 | 매 층 | 3개 중 택1 |
| 레어 | 2 | 매 층 | 3개 중 택1 |
| 에픽 | 4 | 2층마다 | 3개 중 택1 |
| 전설 | 5 | 3층마다 | 3개 중 택1 |
| 초전설 | 6 | 4층마다 | 3개 중 택1 |

→ 커먼은 버프 다량 축적, 초전설은 순수 스탯으로 돌파

## 입장권 시스템

### 획득 방법
| 방법 | 수량 | 주기 |
|------|------|------|
| 매일 무료 | 1장 | 일일 리셋 (KST 자정) |
| BP 구매 | 1장 | 50 BP (1일 2장 제한) |
| 배틀 승리 보상 | 1장 | 10승마다 |
| 캠프 정산 보상 | 1~2장 | 라운드당 |
| 구독자 보너스 | +1장 | 일일 |

### 최대 보유
- 최대 5장 (넘치면 소멸 아닌 획득 불가)

## 포켓몬 스탯 반영

던전에서의 내 포켓몬 전투력 계산:

```
기본 스탯 (희귀도 + 종족값)
  × IV 보정 (C: 1.0, B: 1.05, A: 1.10, S: 1.18)
  × 이로치 보정 (일반: 1.0, 이로치: 1.15)
  × 친밀도 보정 (강화 단계별 기존 배틀 스탯 그대로)
  × 버프 누적
```

→ **S급 이로치 = 약 1.36배** vs 일반 C급
→ 10층 이상 차이 = 랭킹에서 결정적

## DB 스키마 (안)

```sql
-- 던전 입장권
ALTER TABLE users ADD COLUMN dungeon_tickets INT NOT NULL DEFAULT 1;
ALTER TABLE users ADD COLUMN dungeon_tickets_bought_today INT NOT NULL DEFAULT 0;
ALTER TABLE users ADD COLUMN dungeon_best_floor INT NOT NULL DEFAULT 0;

-- 던전 런 기록
CREATE TABLE dungeon_runs (
    id SERIAL PRIMARY KEY,
    user_id BIGINT NOT NULL REFERENCES users(user_id),
    pokemon_instance_id INT NOT NULL,
    pokemon_id INT NOT NULL,
    pokemon_name TEXT NOT NULL,
    is_shiny BOOLEAN DEFAULT FALSE,
    iv_grade TEXT,
    floor_reached INT NOT NULL DEFAULT 0,
    buffs_json JSONB DEFAULT '[]',
    enemies_json JSONB DEFAULT '[]',   -- 만난 적 기록
    bp_earned INT DEFAULT 0,
    started_at TIMESTAMPTZ DEFAULT NOW(),
    ended_at TIMESTAMPTZ,
    season_key TEXT  -- '2026-W12' 같은 주간 키
);

-- 주간 랭킹 (시즌)
CREATE INDEX idx_dungeon_runs_ranking
    ON dungeon_runs(season_key, floor_reached DESC);
```

## 기대 효과

1. **DAU 상승**: 일일 무료 입장권 → 매일 1회는 들어옴
2. **캠프 활성화**: 이로치 전환 수요 → 조각 수요 → 캠프 참여
3. **육성 동기**: IV S급, 이로치, 친밀도 MAX = 고층 필수
4. **커먼/레어 가치**: 버프 많이 받아 역전 가능 → 다양한 포켓몬 키움
5. **리플레이성**: 랜덤 적 + 랜덤 버프 = 매번 다른 경험
6. **소셜**: "몇 층 갔어?" 채팅방 화제, 랭킹 경쟁
