## 주의사항
- InlineKeyboardButton 라벨에는 커스텀 이모지 HTML 사용 불가 → 기본 이모지 유지
- query.answer() 팝업도 커스텀 이모지 불가
- 커스텀 이모지 사용 시 반드시 `parse_mode="HTML"` 필요
- 텔레그램에서 텍스트 복사 시 커스텀 이모지는 fallback 텍스트로 바뀜
- TGS 업로드: 512x512 원본 그대로 업로드 (80% 스케일 불필요)
- `upload_champion_stickers.py` 스크립트로 스티커셋 관리 가능
